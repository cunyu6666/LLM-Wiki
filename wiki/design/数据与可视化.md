# 数据与可视化

> 16 moonvy 设计素材周刊条目

## Overview

数据可视化 16 条，覆盖图表组件、大屏素材和生成式 UI 交互。亮点：**生成式 UI 复刻**（op7418 复刻 Claude 的生成式 UI 交互，AI 在聊天中流式输出交互式图表）、**年度报告设计库**（2025app.designstroll.space 收集 58 个 App 的 800+ 张年度报告截图）、**大屏可视化素材**（DataV 风格标题边框组件）。

数据可视化是 AI 产品界面的重要组成部分——AI 输出的结构化数据需要图表来呈现。与 [AI设计工具](AI设计工具.md) 中的"生成式 UI"方向直接关联。
