## Spline 轻量 3D 工具

> 编译自抖音短视频 7 条 + Moonvy/行业案例交叉引用

Spline（[spline.design](https://spline.design/)）是面向设计师的**轻量化在线 3D 工具**，定位介于 Figma 与 Blender/C4D 之间：无需传统 3D 基础，通过拖拽基础形状（椭圆、圆柱等）+ 右侧参数面板（材质、圆角、厚度）即可快速产出立体 icon、IP 角色和网页动效。网页端在线编辑或下载客户端均可，**免费可用**，已推出官方汉化版。

### 核心工作流

1. **建模**：左侧图层 + 顶部工具栏画形状，右侧调材质/颜色/圆角/挤压
2. **交互**：状态机式交互（如置换层 7→25 做形变）、点光源跟随鼠标、相机缩放
3. **渐变球体**（UI 设计师标配）：球体 400px + 渐变材质 Phenior02 + 置换层乒乓循环 1.5s 缓入缓出
4. **官网改版案例**（Neo Data）：Figma 画原型 → 导入 Spline 加交互 → 两周完成 AI 落地页改版

### 导出与集成

| 场景 | 方式 |
|------|------|
| 零代码网页 | Spline 导出 URL/Embed → Typedream 粘贴到 Link 或 Code 栏 |
| 前端落地 | 复制 Embed URL → Framer（如 Dana Portfolio 模板）InBad 嵌入 |
| iOS 原生 | 导出 3D 嵌入格式 → 粘贴到 Xcode 项目，Spline Metal 渲染器支持 iOS/iPadOS/macOS/visionOS |
| 设计工具链 | Webflow 插件导入、与 Rive 并列用于 Framer/Webflow 项目（见 Lander Studio 案例） |

### 生态扩展

- **Spline Hana**：2D 交互设计文件，补充 3D 生态；支持渐进模糊（可交互，区别于 Figma 静态模糊）
- **AI 功能**：自然语言创建/编辑 3D 模型、AI Style Transfer
- **社区**：类似 Figma Community 的作品分享与灵感获取
- **同类工具**：Vectary（时间轴交互）、Three.js（开发向）
