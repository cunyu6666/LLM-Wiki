# 字体

> 101 moonvy 设计素材周刊条目
## Sources
- Moonvy 月维 设计素材周刊 213 期 (2022-2026)

## Overview

字体资源 101 条，从免费开源字库到排版工具到跨平台兼容方案。几个值得关注的方向：**免费开源字体爆发**（FreeFaces Gallery、Fontshare 等平台让商用免费字体选择大幅增加）、**跨平台字体兼容**（苹方 Windows 版解决了 Mac→Windows 设计稿字体错乱的老问题）、**排版工具进化**（LiftKit 用黄金比例自动优化 UI 间距和字号）。

有趣的是"颜文字库"这类社交装饰字体的出现——字体不再只是排版工具，也成了个人表达的载体。与 [设计系统](设计系统.md) 相关：Design Tokens 中的字体定义是设计系统的基础层。

### 苹方字体 — Windows 版完美兼容 Mac

> 来源：周刊 #192 · 2025/11/30 · [https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/192/](https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/192/)

在跨平台协作中，设计文件最常见的问题之一就是 **字体不兼容**。尤其是 macOS 自带的 **苹方字体**，Windows 默认并不支持，往往会导致界面错乱、排版变形。

现在，你可以通过安装这个 **Windows 兼容版苹方字体包** 来彻底解决问题。只需在 Windows 设备上下载安装即可正常显示设计稿中的苹方字体。

**注意：** 安装前务必先卸载系统中已有的同名字体，避免因为字体重名导致混乱或替换异常。

[https://ai.feishu.cn/wiki/XKsGwDUHqiEENAkhZIwcdRR0ngz](https://ai.feishu.cn/wiki/XKsGwDUHqiEENAkhZIwcdRR0ngz)

### Mac 15 系统字体 PingFangSC 丢失

> 来源：周刊 #135 · 2024/10/07 · [https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/135/](https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/135/)

升级 Mac 15系统之后，使用过程中可能会遇到

- PingFang SC 字体丢失问题，在系统「字体册」激活并下载 PingFang 相关字体，然后刷新或者重启设计软件。

- 中文界面有乱码显示，在系统语言设置中，将系统语言更改为「英文」，再切换回「简体中文」，然后刷新或者重启软件就好了。

常见问题：[https://moonvy.com/blog/post/2022/Q&A/](https://moonvy.com/blog/post/2022/Q&A/#mac-15-%E7%B3%BB%E7%BB%9F%E5%AD%97%E4%BD%93-pingfangsc-%E4%B8%A2%E5%A4%B1) (保存链接，持续更新中)

### B612 开源字体系列

### 微软宣布 Aptos 字体将取代 Calibri 成为 Office 默认字体

> 来源：周刊 #074 · 2023/07/23 · [https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/074/](https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/074/)

[https://www.gcores.com/articles/168433](https://www.gcores.com/articles/168433)

15 年来， Calibri 一直是 Microsoft 的默认字体。随着时代的发展高分辨率屏幕的增多，微软开始着手准备适用于 Microsoft 365 的新字体。最终微软从 Bierstadt、Grandview、Seaford、Skeena 和 Tenorite 五种字体中选择了 Bierstadt 作为新的 Microsoft 365 默认字体，并同时将此种字体更名为 Aptos。

与 20 世纪中期的瑞士字体类似，Aptos 是一种无衬线字体，其具有简单的字母和笔画形式，并且易于阅读。Aptos 由不同的几何形状组成，在 Aptos 中小写字母 l 将有一个独特的尾部，将其与大写字母 I 分开。i 和 j 的头部是圆形点，而不是正方形。6 是单笔画，8 则是两个堆叠的椭圆形。

Aptos 字体由设计师 Steve Matteson 设计，他之前还设计了原始的 Windows TrueType 核心字体。 Steve Matteson 将他设计的字体从 Bierstadt 更名为 Aptos 是为了纪念他最喜欢的一个加州小镇。

Office 目前的默认字体 Calibri 是随 Office 2007 发布的，取代了更早的 Times New Roman。彼时为了配合 ClearType 渲染技术，Calibri 也是作为微软委托创作的多个字体之一并最终被选中。

### 思源字体（Noto）的命名故事

作为完全免费且全球通用的思源字体系列，隶属于Google的Noto字体家族。Noto的英文全称是"no tofu"，tof在英文里是"豆腐"的意思。之所以这么命名，是因为早期在互联网上很多无法显示的字体会在网页上形成一个个方框，很像"豆腐块"。因此Google决定创造一款字体来消灭这些网页上的豆腐块。

### 三种毛笔字制作方法

**拼法**：找一个垫底字，拿着现有笔触拼接。问题：如果笔触种类不多，做的毛笔字比较单一，需要多收藏毛笔笔触。

**写法**：在手绘板上用笔刷先找感觉，横竖撇那折重一点下笔、轻一点下笔，最后找一个垫底字疯狂描写。写的过程中试重下笔、轻滑、重收尾的感觉，多适应一两天必学会。

**拍法**：不会写毛笔字也没关系，多准备几个笔（毛笔、秀丽笔），在纸上写完后手机拍照导入PS，曲线色阶一起上让白的更白、黑的更黑，再导入AI描摹变成矢量图。

### 拉伸字体

一直以为拉伸字体是用鼠标拉变形，其实有专门的拉伸字体可以直接使用。在字母后面多打几个同样的字母，字体就会自动变长。不管是用在海报还是潮流设计中都非常有设计感。
