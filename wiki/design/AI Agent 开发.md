## Overview

AI Agent 是 2025-2026 年最活跃的技术方向，正在从概念验证走向产业落地。Karpathy 在两小时访谈中直言"未来十年没有 AGI，只有 Agent"，Anthropic 和 Databricks 的 CEO 对话也达成了共识："Agent 产品，快者为王"。

从技术栈看，Agent 开发经历了三个阶段：**单 Agent + Tool Use**（ReAct 范式）→ **Multi-Agent 协作**（2025 年主流，但 Anthropic 2026 年初宣称"去年的 Multi-Agent 全是假的"）→ **Agent-as-a-Service**（托管式 Agent，Anthropic Harness 发布）。执行层也在从 Bash 脚本转向 TypeScript（"Bash 不够用了，AI Agent 的执行层开始转向 TypeScript"）。

OpenClaw（小龙虾）是本阶段最具话题性的 Agent 产品——从爆火到被谷歌制裁、Karpathy 怒喷、团队版开源，它的起落折射出 Agent 生态的野蛮生长。Notion 发布 Custom Agents、Goose 本地 Agent 框架、字节 Harness Agent 等则代表了不同路径的探索。

对 UX 设计师而言，Agent 带来的核心问题是：当 AI 可以直接"做事"而非"展示结果"时，界面设计的角色是什么？Notion 创始人 Ivan Zhao 的判断是："不能被 Agent 用的产品没有未来"。
