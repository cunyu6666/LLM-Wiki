## Sources
- Moonvy 月维 设计素材周刊 213 期 (2022-2026)

## Overview

背景素材 67 条，覆盖渐变生成器、矢量图案、纹理壁纸三大类。**渐变工具**已经高度成熟——gradientora.com 提供 1100+ 种渐变方案，一键生成 CSS 代码；**矢量图案**（Toggle UI 配套的网格/点阵/栅格素材）支持 Figma 编辑，适合做产品界面的底层纹理；**AI 生成背景**（Google AI Studio 生成微缩场景）是新趋势。

背景与渐变是 UI 设计的"底色"，与 [UI模板与设计系统](UI模板与设计系统.md) 中的组件搭配使用，与 [材质与拟物](材质与拟物.md) 中的玻璃/金属质感背景有交叉。
