# 图库与配色

> 17 moonvy 设计素材周刊条目
## Sources
- Moonvy 月维 设计素材周刊 213 期 (2022-2026)

## Overview

图库与配色 17 条，是数量最少的素材分类，但每条都很实用。覆盖：**配色参考**（游戏类 GUI 界面配色、黑色主题 UI 设计源文件）、**设计历史**（webdesignmuseum.org 网页设计历史画廊，浏览旧网站发现被遗忘的趋势）、**艺术家作品**（将喜欢的艺术家风格作为 AI 提示词的视觉参考）。

配色是 UI 设计的基础能力，与 [背景与渐变](背景与渐变.md) 中的渐变配色方案、[材质与拟物](材质与拟物.md) 中的材质色调选择都有关联。
