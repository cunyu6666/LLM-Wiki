# 硬件项目

> 4 moonvy 设计素材周刊条目
## Sources
- Moonvy 月维 设计素材周刊 213 期 (2022-2026)

## Overview

硬件项目 4 条，收录与设计交叉的 DIY 硬件和开源项目。虽然数量少但覆盖了从实用工具到概念设计的完整光谱：**实用工具**（ESP32 小屏幕桌面显示 Claude Code 配额、Umbrel 个人家庭服务器）、**概念设计**（变形 Game Boy 掌机融合 Switch 模块化理念）、**开源手表**（墨水屏智能手表 DIY，3D 打印外壳 + ESP32 固件）。

硬件项目展示了设计师跨界的能力——从屏幕内的 UI 延伸到物理产品的交互设计。

### 个人家庭服务器，Umbrel

> 来源：周刊 #067 · 2023/06/04 · [https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/067/](https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/067/)

个人家庭服务器 Umbrel 让你的数据都存在你自己的本地，无需技术技能，在 30 秒内启动并运行。还有一个完整的应用程序商店。网站的设计风格暗色为主，从硬件到软件结合的设计。应用程序中还有 [Penpot](https://penpot.app/) 可以使用。

主页：[https://umbrel.com/](https://umbrel.com/)

应用程序：[https://apps.umbrel.com/](https://apps.umbrel.com/)
