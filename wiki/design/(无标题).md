## 摘要

🌸 桌面便签神器 随手记灵感，这个开源小工具好舒服…
最近看到一个国产开源项目 花笺 Floral Notepaper，咱就是说，它不是那种功能堆满、越用越累的笔记软件，而是主打轻量、优雅、现代化的本地便签工具。官方介绍里写到，它基于 Tauri 2 + React 构建，适合随呼随用。
✨ Markdown 也支持 可以写 Markdown，还支持 GitHub Flavored Markdown，编辑和预览模式能实时切换，记技术片段、待办、灵感都很顺手。
⚡ 随时唤出 默认全局快捷键是 Ctrl + Space，也可以从托盘打开，临时复制文本、记录想法不用再翻半天 App。
🧷 磁贴模式很实用 可以把笔记固定在桌面某处，当作桌面待办、剪贴板、游戏/看视频时的临时记录都挺合适。
一句话：如果你想要一个“打开快、够好看、不臃肿”的本地便签，花笺值得收藏。
