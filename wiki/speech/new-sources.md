## 🔴 S 级 · 直接改变演讲叙事角度的素材

### 1. OpenAI大神教你如何榨干Codex
**核心观点：**
- Jason Liu 把 Codex 改造成可长期运行的自主工作系统
- 跨月存活的巨型线程，项目背景自然沉淀，Agent 具备连续性
- **Heartbeats + @computer**：Agent 每 30 分钟扫 Slack/Gmail，起草回复但不发送，由人决定是否发出
- 洗澡前让 Codex 盯着亚马逊客服排队，洗完澡退款到账
- **验证机制**："没有验证机制的野心，顶多算个愿望而已"
- **记忆放在自己手里**：放弃 Codex 内置记忆，用 Obsidian vault 管理所有工作记忆
- "文件系统仍然是最可靠的记忆基础设施"
- 口述下任务，不打字——保留原始思路，不刻意优化 Prompt

**演讲价值：** 这就是"Design → Build"的终极形态——人定义意图和验证标准，AI 持续执行。设计师最擅长的就是定义验证标准（验收条件）。
**路径：** [`Cubox/OpenAI大神教你如何榨干Codex-2026-05-23.md`](../../Cubox/OpenAI大神教你如何榨干Codex-2026-05-23.md)
**建议入库：** `wiki/design/AI编程与Vibe Coding.md` + `wiki/design/AI实操与工具.md`

### 2. 设计走查这个苦活，我终于用AI跑通了全自动
**核心观点：**
- 设计师自己做走查，占日常工作一半时间，纯体力活
- 用 Codex + iPhone 镜像 + Computer Use 实现全自动走查
- **Skill 化**：把走查逻辑写成 Skill，一次编写永久复用
- "AI 帮你跳过从 0 到 6 的部分，从 6 到 9 还是得你自己来"
- **"AI 帮你量尺寸、比颜色、数像素，这些都是尺子的活。但什么时候该打破规范、整体调性对不对，只有设计师自己知道"**
- 日本词"目利き"——一眼看出好坏的眼力

**演讲价值：** 完美案例！设计师用 AI 消灭自己最痛苦的体力活，把时间还给创造性工作。而且这篇文章的作者也是设计师，用 figma+MCP+codex 工作流。
**路径：** [`Cubox/设计走查这个苦活，我终于用AI跑通了全自动-2026-06-01.md`](../../Cubox/设计走查这个苦活，我终于用AI跑通了全自动-2026-06-01.md)

### 3. Agentic Workflow：AI重塑了我的工作流
**核心观点：**
- 吴恩达 4 种 Agentic Workflow 设计模式：Reflection / Tool Use / Planning / Multi-Agent
- **"Reshape your workflow with AI? or Reshape your AI workflow?"**
- 人机协同 3 种模式：Embedding（人做大部分）→ Copilot（协同）→ Agent（AI 做大部分）
- "输入→处理→输出→反馈"四个齿轮的信息处理模型
- **行业 Know-how 是冰山下的不可被观测的竞争力**——短期内不可被替代的固有知识资产
- "曾被认为异想天开的想法，都可能是 AI Agent 的未来"

**演讲价值：** "行业 Know-how 是冰山下的竞争力"这个观点直接支撑演讲——设计师多年积累的人机交互经验就是那个冰山。
**路径：** [`Cubox/Agentic Workflow：AI重塑了我的工作流-2026-06-03.md`](../../Cubox/Agentic%20Workflow：AI重塑了我的工作流-2026-06-03.md)

### 4. HTML版剪映！Open Design 团队 3 天写了 3 万行代码
**核心观点：**
- 3 天时间、小团队，用 AI Agent 写了 3 万行代码做出完整视频制作工具
- 基于 Hyperframes（HTML+CSS+GSAP），把 HTML 变成视频
- 自动识别本地 14 种 Code Agent（Claude Code、Cursor、Codex 等）
- 一句话描述 → Agent 选模板 → 填内容 → 渲染 MP4
- 分页预览、分页编辑、帧文字编辑
- 已接入 MiniMax 背景音乐和旁白

**演讲价值：** "3 天 3 万行"就是"小团队做大事"的极端案例。而且这个工具本身就让设计师能直接做视频——不需要专业剪辑师。
**路径：** [`Cubox/HTML版剪映来了！…-2026-06-08.md`](../../Cubox/HTML版剪映来了！Open%20Design%20团队最新开源力作，3天时间，写了3万行代码！-2026-06-08.md)
