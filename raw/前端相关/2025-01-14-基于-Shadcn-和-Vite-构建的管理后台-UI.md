# 基于 Shadcn 和 Vite 构建的管理后台 UI

[mp.weixin.qq.com](https://mp.weixin.qq.com/s?__biz=MzI4NjM0NjA3Ng==&mid=2247486801&idx=1&sn=42ab7224b2854af3f65670fc33625dbf&chksm=eafe2c6d2119c2e50897ff429bc487ded396f91e613aa8793a048403c48b686d65dc2c0e5119&mpshare=1&scene=1&srcid=0114fnAYruRoe6mSUDXFPZH3&sharer_shareinfo=286cbccf78f0e6a43247be8981073afa&sharer_shareinfo_first=286cbccf78f0e6a43247be8981073afa)CTOLib码库

这是一个使用 Shadcn 和 Vite 构建的响应式、易访问的管理后台 UI。作者在工作和个人项目中积累了丰富的仪表盘 UI 经验，并将其整理成可复用的组件集合。虽然包含一些自定义组件，但部分代码直接参考了 ShadcnUI 的示例。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FzzicjHnpCza2yjLtHZdeuNUNOTCo2jHs8AG9Y3qPL5QFrINjnFJ5gxjiaTEnN9FJvrtevW9z1MvKNPCSlka7jj3A%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

**功能特点：**

•明暗模式切换•响应式设计•可访问性•内置侧边栏组件•全局搜索功能•10 多个页面•额外自定义组件

**技术栈：**

•UI 框架：ShadcnUI (TailwindCSS + RadixUI)•构建工具：Vite•路由：TanStack Router•类型检查：TypeScript•代码规范：Eslint \& Prettier•图标：Tabler Icons

**使用方法：**

1.克隆项目：git clone https://github.com/satnaing/shadcn-admin.git2.进入项目目录3.安装依赖4.启动服务器
