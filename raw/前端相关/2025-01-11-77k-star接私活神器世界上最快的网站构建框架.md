## 核心功能

### 1. 本地开发预览

写完内容后可以实时预览效果：

    hugo server

### 2. 多语言支持

一份内容可以输出多个语言版本,方便做国际化：

    content/
    ├── _index.en.md    
    ├── _index.zh-cn.md
    └── posts/
        ├── hello.en.md
        └── hello.zh-cn.md

### 3. 内容管理

支持Markdown编写,可以设置草稿、发布时间等：

    ---
    title: "文章标题"
    date: 2023-01-01
    draft: true 
    ---

### 4. 主题系统

有大量开源主题可以直接使用，官网发布的有300+主题可供选择使用:

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FXYOz6UPUDZvvRbkxgptweC9f4ibibmSq1JKOtVNSvvYiaeJbb49SXbvud4MetR6fVhsPyNicTOERXpFIlLyF4cebrw%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

也可以自己开发：

    theme = "papermod"

### 5. 资源处理

* • 图片压缩和处理

* • CSS/JS打包压缩

* • Sass编译

* • 支持CDN

## 安装使用

### Windows安装

1. 1. 下载Hugo安装包

2. 2. 解压后添加到环境变量

3. 3. 验证安装：`hugo version`

### Mac安装

    brew install hugo

### Linux安装

    sudo apt install hugo

### 快速上手

1. 1. 创建站点:

2.

       hugo new site myblog

3. 2. 添加主题:

4.

       cd mybloggit submodule add https://github.com/theme.git themes/theme

5. 3. 修改配置:

6.

       # config.tomltheme = "theme"

7. 4. 创建文章:

8.

       hugo new posts/first.md

9. 5. 本地预览:

10.

        hugo server -D

11. 6. 生成网站:

12. 
13.

        hugo
