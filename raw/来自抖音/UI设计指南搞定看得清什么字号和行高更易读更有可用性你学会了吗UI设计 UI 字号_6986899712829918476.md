UI设计指南搞定看得清第一点文字字号虽然视频格式很受欢迎但网络上的大多数信息仍然是文字形式这就是为什么良好的可读性和易读性对于用户体验的可用性至关重要那么如何让界面看得清呢就不得不说到文字的字号和行高首先我们先说一下字号那么这三个界面哪个正文文字你能看得清楚它们分别是14 12 10在移动设备上12 10是很难看清楚的所以在做设计的时候正文的字体大小至少要为14才能保证基本的易读性另一个影响易读性的是航高这三个界面航高分别是1.5 1.4 1.3那基本上1.4 1.3是看不清楚的所以通常航高设置为1.5它给了界面透气的呼吸感多了几分精致
