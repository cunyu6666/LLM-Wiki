## 文字稿

UI动效有哪些基本规则呢如果你是一个喜欢动效的UI设计师那么一定要看这个系列视频首先是动效的持续时长和速度我们可以从四个方面来简要的概括一下第一点则是时长当元素变化时动效应该足够慢让用户注意到变化的发生同时又不能慢到需要用户等待影响用户的操作如果所用的动效是用于装饰或者吸引用户的注意那么请忘记这个规则常规的时间大约可以控制在200-500ms之间低于100ms的动效几乎都是瞬间很难被识别出来而超过一秒的动效会让人有持质感然后是距离与时间的关系与较大的元素相比较小的元素运动速度应该更慢因为相同的移动距离对于小元素而言位移距离与自身大小的比例更大相比偏移更远而大小相同的元素在移动的时候移动距离最短的元素是最先停下来的这一点似乎显而易见但这更多的是针对于不同设备也就是说更大的屏幕则意味着更远的距离更长的时间然后是动效的移动应符合运动规律当元素运动到边界或者是发生碰撞的时候会使用弹跳来对弓撞的能量进行分摊而弹跳的特效在多数情况下是不合适的因为它很容易分散用户的注意力在一些特殊的情况下我们也可以适当的使用另外值得一提的一点是虽然在往期视频我曾提到使用运动模糊能够使动效更加的生动、自然但在UI设计中尽量不要在运动中使用模糊的效果因为UI设计中元素的运动最应该是清晰的
