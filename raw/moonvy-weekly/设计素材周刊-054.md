## 📰 设计资讯

### Open AI 开放 ChatGPT 模型 API

3 月 2 日，OpenAI 宣布在官方博客中开放了 ChatGPT API (在原有 GPT API 中可选 gpt-3.5-turbo 了) ，开发者可以调用它编写应用或者融入自己的应用中，比如自动总结内容、优化文案、编写代码、翻译服务等。这意味着过一段时间各家软件工具都会推出 AI 相关功能了，尤其是笔记工具。

并且相比原有 GTP 模型的 API 价格下降了 10 倍，中文字符的计费方式也优化了，价格最终是 1000 token/0.002 美元 。虽然有人说这是低价抢占市场，不过实际使用后发现这个价格对也不算便宜，因为计费是包含输入和回答的，并且是不保存上下文的也就是说连续回答问题每次回答都得发送全部聊天记录，如果用 API 优化一次本文章（4822 token）得 0.12 元人民币。感觉价格降低到目前的 1/10 后会比较实用，这就要等竞品出来打价格战了。

对于大部分人来说，等着接下来几周大规模的 AI 应用落地吧。

目前已经有几个可以体验使用新 API 的 AI 小工具了：

- [一键总结 哔哩哔哩 视频内容 Powered by GPT-3.5 AI](https://b.jimmylv.cn/)

- [ChatGPT 聊天机器人](https://freegpt.one/)

- [ChatGPT Demo](https://chatgpt.ddiu.me/)

新闻：
