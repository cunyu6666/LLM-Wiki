### FigmaEX 客户端版本更新：125.9.10

## 🏆 产品推荐

### Lotties 文件编辑器

来自国内的开发者，快速修改 JSON 文件中动画图层的颜色，并轻松删除不需要的图层。让动画编辑变得轻松高效，同时上传你的 Json 动画可以将不需要的元素删除重新导出 Json 动画，目前完全免费

投稿人：Alban-白

[https://lottieeditorx.cfd/](https://lottieeditorx.cfd/)

![](https://moonvy.com/blog/_assets/e4731cf6daa44a6d.png)

### LottieLab 水印去除工具

超简洁的 LottieLab 去水印工具，上传你的文件自动去掉水印信息

投稿人：Alban-白

[https://volodymyr-malishevskyi.github.io/lwrt/](https://volodymyr-malishevskyi.github.io/lwrt/)

Lottie 类型的动画文件无法本地预览效果，可以将 JSON 动画文件上传到月维中进行预览，发给团队或者开发者，开发者可以在月维进行资源下载和使用

![](https://moonvy.com/blog/_assets/eb9a5a248537a391.png)
