### Goodpalette -  UI 设计在线配色

### ClipDrop - 图像智能放大工具

使用 AI 将图像清晰度变高、降噪图像、锐化模糊边缘等 2倍免费，还提供了 API 集成。类似的工具还有开源的 [waifu2x](http://waifu2x.udp.jp/)。

[https://clipdrop.co/image-upscaler](https://clipdrop.co/image-upscaler)

![](https://moonvy.com/blog/_assets/c8c9c9b13b57563d.png)

### Preline - 开源的 CSS 组件库

拥有超过 250 个 UI 组件，包括卡片、按钮、下拉菜单、导航栏、模式、表单等。配套的 UI 设计文件网站上说快上线了，这套 UI 库还是值得期待的。

[https://preline.co/](https://preline.co/)

![](https://moonvy.com/blog/_assets/b2b936b0997be186.png)

### 用 Edge 浏览器实现电脑与手机互传文件

Microsoft Edge Drop – 是 Edge 浏览器的新功能，它可以让 Edge 浏览器通过 Onedrive 空间同步文件与文本，支持在电脑与电脑，或者电脑与手机间同步文件，方便快捷。

目前的 Drop 功能需要 Dev 或者 Canary 版本，还未加入正式版本，并且也需要登录 Live 账号，因为同步使用的是 OneDrive 空间。

[https://www.appinn.com/microsoft-edge-drop-dev/](https://www.appinn.com/microsoft-edge-drop-dev/)

![](https://moonvy.com/blog/_assets/43290c1f313b323b.png)

### 一个新的在线压缩图像工具 - shrink.media

压缩图像的文件大小，在网站或程序里使用图像尺寸越小加载速度越快。

类似工具有很多，他们都会把你的图片上传的他们的服务器上进行压缩，如果你对图片隐私比较在意，可以使用我们的在线压缩工具：[像素蒸发](https://moonvy.com/apps/PxEvapo/)，它仅在你的浏览器上工作，不会上传任何数据。

[https://www.shrink.media/](https://www.shrink.media/)
