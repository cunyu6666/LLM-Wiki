### 即时 AI 内测版上线

国产设计工具纷纷公布其 AI 计划，即时设计的「即时 AI」开始内测版使用。通过文字描述可以生成 UI 设计图 和 原型图。整个过程操作很简单易用，输入文字稍等片刻，分图层的设计图就出来了。

就对 UI 设计的可用性而言，还是有待提高，没法达到在工作中使用的程度，不过也许在做早期原型之类的场景会有用处。作为为数不多现在可以体验的 UI 方向的 AI 工具还是很值得亲自体验一下的。

[https://js.design/ai-upcoming](https://js.design/ai-upcoming)

![](https://moonvy.com/blog/_assets/f105260e5bce9e67.png)

### 阿里版 ChatGPT ：通义千问

国内[百度文心一言](https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/056/#%E6%96%87%E5%BF%83%E4%B8%80%E8%A8%80%E5%8F%91%E5%B8%83)之后，阿里版的类 ChatGPT “通义千问”也上线了。**通义千问** 是由阿里[达摩院](https://damo.alibaba.com)研发的 AI 聊天机器人，具备多轮对话、知识搜索、内容创作、表达观点、撰写代码等能力。

目前产品是邀请制，要想了解水平究竟如何，可以看看这个评测视频：[https://www.bilibili.com/video/BV1os4y1N7V5/](https://www.bilibili.com/video/BV1os4y1N7V5/)

官网：[https://tongyi.aliyun.com/](https://tongyi.aliyun.com/)

### B 站推出「高能链」App 用以管理用户数字资产

### Midjourney 推出 Web 版本

不再依赖 Discord 里生成图像， Midjourney 悄悄推出了自己的 Web 版本。目前只有生成过 10000 张以上的账号可以参与内测。想了解具体 Web 版的样子可以看视频：[https://www.bilibili.com/video/BV1B84y1T72r/](https://www.bilibili.com/video/BV1B84y1T72r/)

想了解自己生成过多少张可以输入`/info`，回车后会出现你的生成信息。如果你超过 10000 张就会出现 `#10000-club` 的频道，进入该频道就可以参加 Midjourney Web 的使用了。

[https://alpha.midjourney.com/app/imagine/](https://alpha.midjourney.com/app/imagine/)

![](https://moonvy.com/blog/_assets/cbb5931a483469c3.png)

### 提示词可视化编辑器 🥣 OPS/OpenPromptStudio

我们的设计师最近也常常用 AI 来辅助做图，不过在使用各种 AIGC 工具时发现编辑提示词（Prompt）不太方便，所以我们做了一个小工具（开源免费）来解决提示词编辑过程中遇到的问题：

- 显示英文提示词的中文翻译 - 翻译输入的中文提示词到英文（因为 Midjourney 仅支持英文提示词）

- 为提示词进行分类（普通、样式、质量、命令）

- 轻松的排序、隐藏提示词

- 把提示词可视化结果导出为图片

- 常用提示词词典

- 通过 Notion 管理提示词词典

小工具链接：[https://github.com/Moonvy/OpenPromptStudio](https://github.com/Moonvy/OpenPromptStudio)

视频介绍：[https://www.bilibili.com/video/BV15N411P7D3](https://www.bilibili.com/video/BV15N411P7D3)

![](https://moonvy.com/blog/_assets/3ad3b494331b4f95.png)
