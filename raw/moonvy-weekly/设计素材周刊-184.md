## 📰 设计资讯

### Figma Make 复制为设计图层

Figma 最近上线了新功能“复制设计”功能，将 Figma Make 预览转换为可以在 Figma Design 中粘贴和编辑的设计图层。

在 Figma 中 AI 生成 UI 之外又增加了 Make 生成 UI 的方法。Figma Make 是付费版和新认证的教育版用户都可以使用的

[https://www.figma.com/release-notes/](https://www.figma.com/release-notes/)

![](https://moonvy.com/blog/_assets/c41ef666decf38c7.png)

### Figma MCP 新增3个Beta功能

Figma MCP 服务器和 Code Connect 推出三项 Beta 功能：远程访问、检索 Make 资源，以及全新 Code Connect UI，可一键连接 GitHub，把 Figma 组件更轻松映射到代码库。

[https://www.figma.com/release-notes/](https://www.figma.com/release-notes/)

### Framer 推出全新「设计模式」

Framer 推出全新 「[Design Pages](https://www.framer.com/events/)」，让它从网页构建器进化为真正的设计工具。可以在自由画布上使用**矢量编辑、P3 颜色、图像导出、高级蒙版等完整工具进行探索和实验**，并可一键将设计转为网页，轻松添加断点。Wireframer 也能并用，快速生成独特布局。最棒的是，设计页面完全免费，项目与页面不限量。

Framer 开始发力设计模式了，这些年 Framer 产品了做了多次转型，从最开始做原型、网页构建工具，如今迈出了设计工具这一步

[https://www.framer.com/updates/design-pages](https://www.framer.com/updates/design-pages)

![](https://moonvy.com/blog/_assets/b12bd741b2f3f7f0.png)

### Google Labs 推出 Mixboard

这是一款实验性的 AI 驱动概念画板。可以将参考图、生成结果都展示在自由画板中，并且多次选中多次生成，帮助完善您的想法，并使用最新的图像生成模型`Nano Banana`提供支持，现已推出仅限美国地区的公测版！

[https://labs.google.com/mixboard/projects](https://labs.google.com/mixboard/projects)

![](https://moonvy.com/blog/_assets/f2323674a8dfbf1.png)

### 创意网站设计欣赏

鼠标可以控制交卷滚动的方向和位置，展开后会有预置的图片展示，这个效果适合展示大量图片信息时使用

网站：[https://www.laxspace.co/](https://www.laxspace.co/)

非常精彩的幕后思考和故事：[https://tympanus.net/codrops/2025/09/23/lax-space-designing-with-duct-tape-and-everyday-chaos/](https://tympanus.net/codrops/2025/09/23/lax-space-designing-with-duct-tape-and-everyday-chaos/)

**设计原则**

随着胶带在屏幕上不断展开，画面很容易显得混乱和沉重。为了平衡这种能量，界面被刻意保持简洁干净。目标是让视觉效果成为舞台的主角，同时留出充足的留白，让用户能够自在探索和玩耍。

体验中还编织了一层互动感。动画会对用户操作做出回应，营造出运动与交互的感觉。隐藏的小细节，比如可以回放、围绕元素旋转，或是一个闪烁的小点，用来提示未被发现的项目。

按下空格键即可回放胶带，让它重新绘制一条路径。
