## 🏆 产品推荐

### Figma 圆角计算插件

插件 Smoother 自动计算父框架和子框架之间的完美半径关系，类似 Sketch 前段时间发布名为“同心角”的功能

[https://www.figma.com/community/plugin/1533519971089532307/smoother](https://www.figma.com/community/plugin/1533519971089532307/smoother)

![](https://moonvy.com/blog/_assets/482c6ca9830538bc.png)

### 基于React 的光标动画库

使用React、TypeScript、Tailwind CSS 和 Framer Motion 构建的开源动画光标组件，可以让你的网站更有特色

[https://cursify.vercel.app/](https://cursify.vercel.app/)

![](https://moonvy.com/blog/_assets/1e6e36caef342e8e.png)

### 2款开源的「表情符号选择器」组件

使用 Tailwind 和 React 制作的快速、可组合、无样式的表情符号选择器

[https://emoji.ferrucc.io/](https://emoji.ferrucc.io/)

[https://frimousse.liveblocks.io/](https://frimousse.liveblocks.io/)

![](https://moonvy.com/blog/_assets/90a35db94314e49b.png)

## 🎊 设计素材

### 网站着陆页背景样式

提供非常好看的背景的代码样式库，可以用做网站着陆页背景，直接拷贝代码就能使用。包含渐变、图案、多色晕染等效果类型，点击预览立即生效能够非常直观的预览。网站内容全部免费

[https://patterncraft.fun/](https://patterncraft.fun/)

![](https://moonvy.com/blog/_assets/8bdca9afe74ad07b.png)

### 实用好看的「选中效果」代码组件

用于选择和突出显示文本的轻量级且可组合的 React 组件，完全免费在你的项目中使用

[https://lisere.ankur.design/](https://lisere.ankur.design/)

![](https://moonvy.com/blog/_assets/7b6648771935120d.png)

### 一套金属质感的按钮

美观的金属按钮组件，包含触觉反馈，支持一键复制代码使用

[https://button.lakshb.dev/](https://button.lakshb.dev/)

![](https://moonvy.com/blog/_assets/9298e61c183d2d20.png)

### Apple OS 26 设计源文件

现在 Figma 的社区中已经能够看到全部的 OS 26 各个平台的 UI 设计规范文件了

[https://www.figma.com/@apple](https://www.figma.com/@apple)

![](https://moonvy.com/blog/_assets/89695307cc363b2f.png)
