## 📰 设计资讯

### 强生公司放弃了 136 年历史的 LOGO

强生公司放弃了原来使用了 136 年的草书字体风格的 LOGO，而采用了更加便于阅读的人文主义字体。感觉有些可惜，新 LOGO 虽然更现代，但并不如原来的 LOGO 有辨识度，只是一个泯然众生的普通 LOGO 了。

[https://www.rologo.com/johnsonjohnson-abandoned-its-136-year-old-logo-and-renamed.html](https://www.rologo.com/johnsonjohnson-abandoned-its-136-year-old-logo-and-renamed.html)

## 🏆 产品推荐

### Lottielab 在线动画编辑器

Lottie 在线编辑器 Lottielab，支持导入 Lotties、SVG 或从 Figma 导入，制作时间轴动画。支持导出 Lottie、Gif 或 MP4。

[https://www.lottielab.com/](https://www.lottielab.com/)

![](https://moonvy.com/blog/_assets/104bc90ecaa4508f.gif)

### generative-dynamics 通过静态图片生成可交互动态图片

一篇新论文带了了一个有趣的演示，可以用单个图像生成一个可以交互的动态图片，比如风中摇动的花朵与蜡烛。网站还提供了一个可以操作的例子，拖拽修改物体运动的方向。

[https://generative-dynamics.github.io/#demo](https://generative-dynamics.github.io/#demo)

![](https://moonvy.com/blog/_assets/8d82d4d28769149e.gif)

### v0  - Vercel 推出的 AI 驱动网页设计与开发工具

可以根据提示词生成网页设计与对应的代码（它基于 Shadcn UI 和 Tailwind CSS 的复制粘贴友好的 React 代码），并且可以针对生成的网页的一部分再次用提示词修改。目前在内测中，可以加入候选列表

[https://v0.dev/](https://v0.dev/)

![](https://moonvy.com/blog/_assets/e156df33bf769fa8.png)

### Microsoft Design SDXL 新模型

Stable Diffusion Bdicon模型的作者发布了新模型：Microsoft Design SDXL，使用文字描述就能生成效果比较好的模型图片，比之前的生成门槛更低但生成效果更好。

[https://www.liblib.ai/modelinfo/823d796ea21548b794bdedadfddb0747](https://www.liblib.ai/modelinfo/823d796ea21548b794bdedadfddb0747)

![](https://moonvy.com/blog/_assets/6386f1d766b2b5d0.png)

### Figma 3D 镭射效果插件：Shapes 3D

国内 3D 插件作者 [@Scriabin](https://www.figma.com/@scriabin)，又发布了一款新插件，文本、矢量图形可以生成具有反射的3D效果。

[https://www.figma.com/community/plugin/1266663542116780385/Shapes-3D](https://www.figma.com/community/plugin/1266663542116780385/Shapes-3D)

![](https://moonvy.com/blog/_assets/ff0ece95652371ce.gif)

### Figma 自动删背景插件：Remove Background

插件完全免费，选择图片启动插件稍等片刻就可以自动删除背景，非常轻量的抠图插件。

[https://www.figma.com/community/plugin/1278756064864777232/Remove-Background-(Privacy-First)](https://www.figma.com/community/plugin/1278756064864777232/Remove-Background-(Privacy-First))

### 汽车中控屏幕查询网站

该网页包含了车辆中控屏幕中遇到的各种情况的界面截图，包含方向盘、座椅、蓝牙、天气、浏览器、辅助驾驶、信息娱乐设置、输入方式、导航等等很细节和全面，作为一个设计查询手册非常棒。
