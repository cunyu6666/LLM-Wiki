### OpenAI GPT-5 大量生成示例

OpenAI 专门在 GitHub 上开源了一个仓库，分享了由 **GPT-5** 一次性生成的大量编程示例。这些示例涵盖了网站搭建、前端应用、小游戏、交互式 UI 等多种类型，全都仅凭一条 Prompt 完成。

同时，所有使用的提示词（Prompt）也在仓库中公开，非常值得学习和参考。你也可以通过 **Codex CLI** 或 **ChatGPT** 来实际体验生成效果。

[https://platform.openai.com/docs/guides/latest-model](https://platform.openai.com/docs/guides/latest-model)

GitHub：[http://github.com/openai/gpt-5-coding-examples](http://github.com/openai/gpt-5-coding-examples)

极简的在线绘图实例：[https://gpt5-coding-examples.vercel.app/online-whiteboard](https://gpt5-coding-examples.vercel.app/online-whiteboard)

![](https://moonvy.com/blog/_assets/4d2f64207d7060f8.png)

### Markdown 编辑器：Modern MD Editor

Modern MD Editor 是一款面向创作者与内容团队的现代化 Markdown 工具，特点：所见即所得编辑 + 实时预览 + 一键复制为公众号/社交平台高保真富文本。支持桌面/平板/手机视口切换，并在复制时自动内联样式，实现字体、行高、字距与主题的统一，降低排版走样风险。

[https://mmdeditor.boxtech.icu/](https://mmdeditor.boxtech.icu/)

![](https://moonvy.com/blog/_assets/25ab82ebf26d801f.png)

### 手机操作系统概念设计

**AmeOS 16** 是概念操作系统，界面采用全新的 **Liquid Glass Material（液态玻璃材质）**，在整个 UI 中。尽最大努力，让这个概念系统在设计与性能上都达到极致甚至能在你的手机上流畅运行。

**AmeOS** 是一个完整的 **移动操作系统模板**，专为设计展示与创意探索打造。
