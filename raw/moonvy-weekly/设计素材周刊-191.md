## 🏆 产品推荐

### Ant Design 6.0 正式发布

本次更新聚焦于**技术底层深度优化**，全面采用**纯 CSS 变量**与语义化结构，移除 IE 支持并完美适配 React 18（面向 React 19）；其核心优势在于支持从 v5 **零成本平滑迁移**（无需任何兼容包），同时新增了 **Masonry 瀑布流**等组件，大幅提升了开发体验与性能表现。

[https://github.com/ant-design/ant-design/issues/55805](https://github.com/ant-design/ant-design/issues/55805)

![](https://moonvy.com/blog/_assets/8dd4e96a99058ac9.png)

### 文件传输工具

这是一款**免费开源**的文件传输工具，使用 **P2P 点对点技术**实现文件的直接传输，无需经过云端服务器；它提供**端到端加密**保护并确保用户隐私（无需个人信息），支持 Windows、Linux 和 macOS 系统，以及支持包括中文、英文等多种语言。

[https://github.com/tonyantony300/alt-sendme](https://github.com/tonyantony300/alt-sendme)

![](https://moonvy.com/blog/_assets/2881c521bf6c6a.png)

### Canned Style - 风格罐头更新

这是一款**高效的 Figma 样式插件**，内置了**上百种**精选的填充、渐变、描边及投影样式；它支持**一键点击复用**，让设计师无需繁琐调整参数，即可极速创建出高品质的按钮、背景等视觉效果，大幅提升设计产出效率。

[https://www.figma.com/community/plugin/1392338579595619092/canned-style](https://www.figma.com/community/plugin/1392338579595619092/canned-style)

![](https://moonvy.com/blog/_assets/55f93ac076d53e7b.png)

## 🎊 设计素材

### Yapraticon 图标库

Yapraticon 中包含 **7500 个图标**的庞大图库，包含 3 种填充风格，支持完全编辑。

[https://www.figma.com/community/file/1571945921897860467](https://www.figma.com/community/file/1571945921897860467)

![](https://moonvy.com/blog/_assets/447f74c00834361f.png)

### Nuxt UI 模板

这是一套**官方免费推出**的 Nuxt UI v4 生产级模板合集，包含**落地页、文档站、作品集和仪表盘**四种核心布局；这些此前属于 Pro 付费版的设计现已全面开放，完美适配 Nuxt UI v4 组件库与 Tailwind CSS v4 架构，支持**全端响应、亮暗模式切换及 Figma 自动布局**

[https://www.figma.com/community/file/1572203339856768251](https://www.figma.com/community/file/1572203339856768251)

![](https://moonvy.com/blog/_assets/359760600688531d.png)

### Balwin x Hyvä - 组件库

为了解决原有 Hyvä UI 库在复杂项目中的局限性，Baldwin 团队开发并**免费开源**了 **The Baldwin x Hyvä 组件库**；这是一套专为**多品牌电商项目**打造的生产级设计系统，采用**原子设计结构**以确保扩展性，并支持**高级变量与模式**以实现轻松的主题定制，完美弥补了原版在灵活性和功能深度上的不足。

[https://www.figma.com/community/file/1572987951693007376](https://www.figma.com/community/file/1572987951693007376)
