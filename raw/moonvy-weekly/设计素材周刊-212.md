## 📰 设计资讯

### Figma Agent 逐步测试版上线

Figma Agent 是嵌入在设计流程中的 AI 智能体，能够基于设计系统自动生成、重组和批量修改界面，帮助设计师完成重复性工作并加速设计迭代

Figma 智能体将在未来几周内逐步推出测试版。测试期间，该智能体不会消耗 AI 点数。正式版发布后，AI 点数将开始计算

请在此处 ([figma.com/join-waitlist](https://www.figma.com/join-waitlist/)) 注册申请提前体验。如果您符合条件并被选中参与测试，我们将会在您加入测试后向您发送电子邮件。加入等待名单并不保证您一定能获得测试资格。该智能体将面向专业版、组织版和企业版套餐的完整用户开放。协作版和开发者版用户可以在草稿中使用该智能体。入门版、教育版和政府版套餐不包含在内。

[https://www.figma.com/release-notes/](https://www.figma.com/release-notes/)

### 2个实用 Base UI 组件库

React、Tailwind CSS 和 Base UI 构建的现代组件库，特点是界面风格现代，组件细节精致，包含大量现成的按钮、表单、弹窗、表格、日历、Drawer 等组件，适合快速搭建 SaaS、后台、AI 产品和现代 Web 应用。相比传统组件库，它更强调视觉质感、动画细节和设计一致性。既能直接复制代码使用，也适合拿来当 UI 灵感库参考

## 🎊 设计素材

### SVG 矢量形状集合

精美的抽象 SVG 形状，只需一键即可立即复制代码。 完美适合想要让界面更动态、更具视觉丰富度的设计师和开发者

[https://spectrums.framer.website/](https://spectrums.framer.website/)

![](https://moonvy.com/blog/_assets/b1cf1b1b052b61a6.png)

### 数学极客风动效库

math-curve-loaders 开源库，完全是用数学公式生成的动效。玫瑰曲线、李萨如曲线这些数学图形，动效极其优雅。纯 HTML+CSS 实现，零第三方依赖，体积极小。 自带可视化调试面板。粒子数、轨迹、速度、颜色，拖拽滑块就能实时预览。 调好参数，一键复制完整代码，粘贴到项目里就能跑。

Github：[https://github.com/Paidax01/math-curve-loaders](https://github.com/Paidax01/math-curve-loaders)

预览：[https://paidax01.github.io/math-curve-loaders/](https://paidax01.github.io/math-curve-loaders/)

![](https://moonvy.com/blog/_assets/88513caa659b9a36.png)

### 动物森友会设计风格组件库（非官方）

Animal 风格的 React 组件库 灵感来源于任天堂《集合啦！动物森友会》游戏界面 ，圆润可爱，眼前一亮。 所有视觉元素、图标和动画都是作者独立设计实现的，安装后一行代码引入样式就能直接用，开箱即用。 除了按钮、卡片这些基础组件，整体配色和字体都还原了游戏里那种清新治愈的感觉。同时提供了在线预览，PC 端和移动端都能体验效果。 不少人用它做了个人博客、儿童英语学习应用、数学练习工具等项目，还衍生出了 Vue、Flutter、Android 版本。 如果你想给自己的个人网站或小项目换个画风，告别千篇一律的 UI，这个库值得收藏试试。

[https://github.com/guokaigdg/animal-island-ui](https://github.com/guokaigdg/animal-island-ui)

![](https://moonvy.com/blog/_assets/f067b55c8ba8f89a.png)
