## 📰 设计资讯

### Figma 布尔运算的快捷键

在Figma UI3的更新中，一级布尔运算功能按钮被移至最右侧面板，引发了众多用户的吐槽。这一改动让许多设计师感到不便，甚至有不少人坚持使用UI2，不愿升级到UI3。然而，Figma团队似乎也意识到了这一问题。为了提升操作便利性，他们在最近的更新中为常用的布尔运算功能增加了快捷键，这或许能在一定程度上缓解用户的不满情绪。

**Mac 系统**

→ 合并选区 `⌥（Option） + Shift + U`

→ 选区相减 `⌥（Option） + Shift + S`

→ 选区相交 `⌥（Optiont） + Shift + I`

→ 选区排除 `⌥（Option） + Shift + E`

→ 拼合选区 `⌘(Cmd) + E`

**PC、Windows 系统**

→ 合并选区 `Alt + Shift + U`

→ 选区相减 `Alt + Shift + S`

→ 选区相交 `Alt + Shift + I`

→ 选区排除 `Alt + Shift + E`

→ 拼合选区 `Ctrl + E`

[https://www.figma.com/release-notes/](https://www.figma.com/release-notes/)

### 微信 Markdown 编辑器

高度简洁的微信 Markdown 编辑器：支持 Markdown 语法、色盘取色、多图上传、一键下载文档、自定义 CSS 样式、一键重置等特性

自动即时渲染为微信图文，让你不再为微信内容排版而发愁！只要你会基本的 Markdown 语法，就能做出一篇样式简洁而又美观大方的微信图文。

在线体验：[https://doocs.github.io/md/](https://t.co/mAjEUGHqfY)

Github：[https://github.com/doocs/md](https://github.com/doocs/md)

### Figma 路径文本插件：Curved Text

[https://www.figma.com/community/plugin/1478676099549634222/curved-text](https://www.figma.com/community/plugin/1478676099549634222/curved-text)
