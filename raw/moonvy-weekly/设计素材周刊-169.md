## 📰 设计资讯

### Figma 即将支持液态玻璃效果

Figma官宣即将推出液态玻璃效果，Figma在X上放出视频预告。上周设计群里讨论AI话题都变少了，回归到设计效果实现， 这次 Figma 的响应很及时👍，其他设计工具暂时还没有动静。

[https://x.com/figma/status/1933624715714965947](https://x.com/figma/status/1933624715714965947)

### Figma 近期更新

- Figma Sites 添加代码级交互和动画

现在，借助代码层，你可以在网站中实时编写和管理 React + Tailwind 动效，也可使用内置 AI 辅助生成。

- 开发模式 MCP 官方服务器测试版上线

Figma Dev Mode 现已支持 MCP 服务器，助力 AI 代理在代码编辑器（如 Cursor、VS Code with Copilot、Windsurf、Claude Code）中实现基于设计的智能生成。

- Apple 字体现已自动可用

在 Figma Editor、Slides 和 Buzz 中，Apple 的 SF Pro 和 SF Compact 字体可自动使用，无需手动安装。FigJam 也支持部分复制粘贴使用。

[https://www.figma.com/release-notes/](https://www.figma.com/release-notes/)

## 🏆 产品推荐

### 设计师社区 Layers 界面改版

原来是一页多图展示，以图为主预览，变成了X一样的设计，更像设计区/帖子的浏览方式。Dribbble 变了之后，不少设计师换到了 Layers 这里分享内容，社区以 UI、UX 作品为主

[https://layers.to](https://layers.to/)

![](https://moonvy.com/blog/_assets/85ed160789069529.png)

### 移除背景工具

超有意思的图像移除背景工具，网站打开像游戏场景，把图片拖到界面中就可以开始移除背景（过程中会把你这张图变成床单丢进洗衣机转几圈）然后就去掉背景可以下载了。🤔这个设定做压缩工具也是可以的吧，哈哈，总之就是很有意思有很实用的工具。

[https://tools.dverso.io/bgremove/](https://tools.dverso.io/bgremove/)

![](https://moonvy.com/blog/_assets/7980431f86f8f44b.png)

### 2个设计师灵感网站

Supahero 与 CTA.gallery 均是高品质的设计作品库，为用户提供“直接可用”的界面设计灵感。

一个专注于按钮与交互行为（CTA），另一个立足网站首屏视觉（Hero Section），共同价值在于：**精选、分类、启发、提升转化**。两者适合在用户界面交互设计的不同关键环节互为补充，是设计流程中不可或缺的灵感来源

[https://www.supahero.io/](https://www.supahero.io/)

[https://www.cta.gallery/](https://www.cta.gallery/)

![](https://moonvy.com/blog/_assets/7eb81eb0b0ba21c.png)
