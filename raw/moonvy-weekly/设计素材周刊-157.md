## 🏆 产品推荐

上周推荐的谷歌多模态 Gemini-2.0-Flash-exp 模型很强，但因为网络问题有些体验门槛，这周推荐几个接入了Gemini-2.0-Flash-exp 模型的产品，打开即用更方便使用。

### 1. Figma 插件：Evoker

该插件支持将自己的 Gemini-2.0-Flash-exp 的 API 填写到插件中，在 Figma 中使用多模态模型。生成之后可以很方便的插入回 Figma

[https://www.figma.com/community/plugin/1483385411494564236/evoker](https://www.figma.com/community/plugin/1483385411494564236/evoker)

![](https://moonvy.com/blog/_assets/aaebfefac4e2eb9d.png)

### 2. Figma 插件 AI-Style Copy

这款插件之前推荐过，插件中包含丰富的 AI 处理工具，最近作者添加了谷歌多模态 Gemini-2.0-Flash-exp 模型，在「AI图像编辑」工具中就能愉快的使用了。不需要填 API 但是有使用次数限制。使用我的邀请码：**REF411VT0TT ，即可获得1天免费使用时间！**，另外在公众号「Moonvy月维」，本期文章的评论区中抽一位幸运读者，赠送三个月的使用激活码。

[https://www.figma.com/community/plugin/1437318459007099882/ai-style-copy](https://www.figma.com/community/plugin/1437318459007099882/ai-style-copy)

![](https://moonvy.com/blog/_assets/eefff83a83228fe8.png)

### 3.在线 AI 创意工具：FLORA AI

集成多种 AI 创意工具的平台，提供文本、图像和视频 AI 模型，最近也新增了 Gemini-2.0-Flash-exp 模型

[https://www.florafauna.ai/](https://www.florafauna.ai/)

![](https://moonvy.com/blog/_assets/49ce94f5b6228db3.png)

### [Figma 插件 Canned Style，新增拟物 [Teenage.engineering](http://Teenage.engineering) 风格样式](#figma-%E6%8F%92%E4%BB%B6-canned-style%EF%BC%8C%E6%96%B0%E5%A2%9E%E6%8B%9F%E7%89%A9-teenage.engineering-%E9%A3%8E%E6%A0%BC%E6%A0%B7%E5%BC%8F)

最近上线的 [OpenAI.FM](http://OpenAI.FM) 的网站 UI 致敬[Teenage.engineering](http://Teenage.engineering)，这个风格一直很喜欢，复刻了一遍添加到 Figma 插件 [Canned Style（风格罐头）](https://www.figma.com/community/plugin/1392338579595619092/canned-style)现在，你就能在设计图中使用它们

[https://www.figma.com/community/plugin/1392338579595619092/canned-style](https://www.figma.com/community/plugin/1392338579595619092/canned-style)

![](https://moonvy.com/blog/_assets/7d6e76d4f2316287.gif)

### 在线 SVG 转 3D 开源小工具

网站支持将任意 SVG 格式的内容生成 3D 模型，支持环境、光照、背景色等选项，适合做简单的 LOGO 或者形状延展，生成内容支持以图片方式导出到本地。

[https://www.vecto3d.xyz/](https://www.vecto3d.xyz/)

![](https://moonvy.com/blog/_assets/e78b224b7d673cfa.gif)

### 腾讯混元 3D 2.0 模型

从图片到模型如此简单，上传3~4个不同角度视图的图片，就能生成真3D模型

在线体验：[https://huggingface.co/spaces/tencent/Hunyuan3D-2mv](https://huggingface.co/spaces/tencent/Hunyuan3D-2mv)

模型：[https://huggingface.co/tencent/Hunyuan3D-2mv](https://huggingface.co/tencent/Hunyuan3D-2mv)
