# Claude"山寨版"来了，支持中文，可配"任意模型"

[mp.weixin.qq.com](https://mp.weixin.qq.com/s?__biz=MzI2MzQ3MzUxMg==&mid=2247487206&idx=1&sn=46d88d209206463670e00d240f83ebcc&chksm=eb0e27e57346530157e588453e70f185c5a0dc56f958737c50409faefe3c1b9dba24057002a2&mpshare=1&scene=1&srcid=0516aEnb3drBjenH8YejKRFS&sharer_shareinfo=f69632f759944bed8fb3f237658c1ae6&sharer_shareinfo_first=f69632f759944bed8fb3f237658c1ae6)Jarvis 甲维斯C


前段时间测试不同模型的 UI 复刻能力，让国内外 AI 克隆了一波 Claude 的官网和桌面版。

**搞完之后想了想，既然UI都克隆好了，那么直接把后端对话能力也接入一下吧。**
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FInUxcnL1BZHCEgmhBrJ6UnWzf1dxL7odxgqxxzgH2WczAGmCxLuFu3TlwUdTIo7JuV4XrBtKOuKuHx6NEZmx0ic2QegvNGjbkGACQFichmGr8%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D0)

经过几天的努力，核心功能已经跑通，特来分享一下软件。（其实很多天之前就搞好了\~\~）

目前 macOS 版和 Windows 已经全部编译成功！

咱们这个"山寨版"可不简单，"**Claude 有的我们未必做，我们有的 Claude 绝对没有！** " 李老板一听，直呼内行！

比如，我们这个版本支持"**纯中文界面** "，Claude 没有吧！

比如，可以切换模型提供商，接入任意国产模型或者国外模型。

比如，我们的软件体积小，内存少，跑得快！
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FInUxcnL1BZGKqqDszc1u1aRBmGv9NCP8mLpMTcQeGyicNbQ7q3hBHLGAlhn9xsmew8Q55UAZaTaHSct2OhgDRw1FDA5QhlsAeHOdWauiaM2hQ%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D1)

官方版 284 MB，我们的只有 2.87 MB，百分之一的体积，赢麻了！

最重要的是，我们不封号！😄 赢了不知道多少次了！

当然，Claude说一句："我的模型是真的"，我们立马就躺下了！

下面就来详细看一下"**李鬼和李逵** "。为了好区分，我们把这个山寨版称为 JClaude 吧！

因为原版软件功能比较多，完全复刻是不可能的。

所以我的思路是先把核心功能搞好，其它功能就先当个摆设吧！

然后我说一下目前有哪些功能是可以用的。

模型选择：
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FInUxcnL1BZFUlT3VPPibAXvMyq8pusVupYrJyibOpGFekibQUvxAo1nYnvFupbYChh487ssicyLucWAT06V2dcmnQlG08lDINCbc3tBJwp58oLU%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D9)截屏2026-04-23 12.08.29

模型选择功能是可以用的，弹出正常。选择不同模型之后，会真的调用不同模型。

对话管理：
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FInUxcnL1BZGWBjpr6QfTOQf8GkSZrMOM5sdPIUdnxYMOVjDWvftsNv6wlqLj2Ch8icPekBxFt2sXa0zafeibC8siaP0Lcs2KcicT3w2ia5UTjHD4%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D10)

*

主题部分包含了深色，浅色，自动（系统跟随）

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FInUxcnL1BZFHc6FLQx2uQ70nVb4ibQ82vspxUJc0aIbkZl0H9cHxDiaw4mcsIw4RXvia3rcn0bI5Q2ebj1oFibFbCREtibuGUaLuY1etWx6WE5sw%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D14)

表面上是 Opus 4.7，实际上可以是任何模型\~\~！

## 3、安装使用

安装非常简单，基本都是双击就好了。

我在不同系统版本的文件里面，都放置了完整的安装说明。
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FInUxcnL1BZFvShxDaqIZgZ5uJRVPIh1J2nhvhyDa3wLRTdHXibZgib2zeaiacysbVvC2eRTUzo3rxzvzE4kuWyIMEsZd5YtYsOib97icjQ2mCeDU%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D15)

macOS 主要注意你的电脑配置，如果是 Intel 系列，记得选 x64，如果是 M 系列选 arm64，如果不知道自己是什么系列，直接选 Universal 这个。

接下来就是正常安装就可以了。

因为我还没有给苹果交"保护费"。所以安装的时候可能因为证书问题有异常提示。这个异常提示是正常现象不要惊慌。

首次安装应该是需要在隐私与安全性里面设置一下了，打开应用之后，去里面找到 Claude，然后点击"仍然打开"就可以了。

具体配置见安装说明！

发布包也自带了 SHA256SUMS 校验文件！

可以使用命令进行校验：

    shasum -a 256 -c SHA256SUMS.txt

看到 OK 表示文件校验通过。

Windows 的话就更加简单了，直接双击 exe，一路下一步，搞定！
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FInUxcnL1BZGVFlzD9gtGslXWQ3RmRHDOmX7CWQiakRyrvtkFicjPaVRZRmUl5Z9ccib3TrC3Via6RKCuxcMkgibibkVmKUibmcHfib6AztSNTMuEMMU%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D16)

**密钥配置**

安装完成之后，唯一要做的事情就是配置一个密钥（API Key）。

如果不配置，直接聊天，就会有如下提示：
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FInUxcnL1BZEpZn5b0iaV9tjAgPXsFkwLHWszoCDAoqT4yHtM7QhKDibu1tAe4RWicj20icEQ5WibWyEjLN0GiaxuT05eWrHN6kEiaJOns7ryJjlTr0%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D17)

配置的方法是，点击左下角的头像，然后选择设置功能。找到模型设置菜单，点击编辑。
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FInUxcnL1BZHo4FDI1doZSOzVI1icdLCZD0dcmOuEZCNQKKWQQSzB6aDEq7GVA7mYZ5T0jm77quXr4jIicHuOXs30cGs1R14ThC2JY1fGx7WOg%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D18)

然后把密钥粘贴到 API Key 这个输入框，最后点击保存就好了。
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FInUxcnL1BZFAf5lCdpVCojCP5PVG968EyicUqAHPTaxedQQ7ibBWESoyCLeZdRGLaCldzzodWo7XLEibTyTt1BNvqF1t5ic67YqH09d36usRDuA%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D19)

这样就可以正常对话了。

如果你没有智谱的API，也可以配置其他提供商。

只要添加提供商，然后配置一下名字、接入点、密钥、模型就可以了。

如果是其他提供商的话，记得修改一下底部的模型映射。

这样就可以轻松搞定了！

这次的项目主要是包含了：

UI 复刻，主题切换，自定义窗口，多语言支持，提供商管理，模型映射，模型接入，对话管理，Markdown 格式支持，多系统支持，多平台发布包制作。

这些功能虽然都不难啊，但是工作量也不小，知识点也不少，细节很多。

所以也花了一丢丢时间，主要是前段时间"村里"断网，Claude也不稳定，耽搁了一些时间。

我再把 JClaude 的网页版搞一搞，然后全部接入正在进行中的词元盲盒（TokenBox）就完美闭环了！

收工收工！ 终于了了一件事儿。
