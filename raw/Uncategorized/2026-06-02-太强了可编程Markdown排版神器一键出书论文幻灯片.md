# 太强了！可编程Markdown排版神器，一键出书论文幻灯片

[mp.weixin.qq.com](https://mp.weixin.qq.com/s?__biz=Mzk3NTA0MTA3MA==&mid=2247484695&idx=1&sn=83eee15961cd88dca23e1a7212d1fde1&chksm=c5027c075bed1ac76d1c2858fecc0540be22d983f41aecb3545fe4906a067551f3410c3137cb&mpshare=1&scene=1&srcid=0602DNp0BY5nD6j2JYgrYPKH&sharer_shareinfo=605b0c29caebff77d7223867565a1a6b&sharer_shareinfo_first=605b0c29caebff77d7223867565a1a6b)Aspheer玩AI

Quarkdown 是一款功能强大的 Markdown 排版系统，它把普通 Markdown 的简洁性与编程语言的灵活性完美融合，让同一份源文件可以直接编译成书籍、学术论文、知识库或交互式演示文稿。

项目基于 CommonMark 与 GFM 扩展，引入了函数、条件语句、循环和变量定义等特性，还提供了丰富的标准库，支持布局构建、数学计算和输入输出操作。用户既能使用内置功能，也可自定义函数与库，轻松实现复杂动态内容。

Quarkdown 支持 HTML、PDF 和纯文本三种输出格式，可一键生成静态网站、分页文档、幻灯片或技术文档。结合实时预览、快速编译和 VS Code 扩展，极大提升了从想法到成品的效率。

GitHub：github.com/iamgio/quarkdown

主要功能：

- 扩展 Markdown 语法，支持函数调用、条件与循环

- 内置标准库，涵盖布局、数学、I/O 等常用操作

- 一键导出 HTML、PDF 或纯文本，覆盖书籍、论文、演示和文档

- 提供项目创建向导、REPL 交互模式和 GitHub Actions 集成

- 兼容 Windows、macOS、Linux，通过脚本或包管理器快速安装
