**HeyGen在数字人方面已经技术垄断了很长一段时间了。**

前段时间**硅基智能开源的HeyGem刚上线** 就给大家推荐过，不知道大家有没有用上。


![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2Fl2VB7h1M5Nawue44MgicNpmJI72q6fwicl3NfjLgIx0WwP94zwJTibVibMGU3sVUHShtTmqjZpZOAuibZIqeT6ggBlA%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

现在已经5.8k星星了。

上次就给大家讲过这个模型的价值，24年年初的时候买一个数字人模型还要**上百万** ，结果硅基智能拿来开源了。

**果然，牛的开源项目一定会让一批闭源的公司倒闭，还会让另外一些闭源的公司利益受损。**  

HeyGen作为受影响最大的产品，竟然直接在GitHub投诉。


![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2Fl2VB7h1M5Nawue44MgicNpmJI72q6fwiclUTuYEcfoTK6NIvdPOiaSziafnkqEiacE6hYpOuXrlSND5G6AEakJAvAag%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

HeyGen的价格是真的贵啊！

据开发者社区消息，**HeyGen以"侵犯商业利益"为由，要求GitHub删除HeyGem项目源码。**

投诉的原因很明显，硅基智能开源的HeyGem确实是动了HeyGen的蛋糕，**一个免费开源的产品跟一个很贵的闭源产品效果差不多，** 用户肯定不愿意去当冤大头了。

**HeyGen的黑历史**  


我觉得还是有必要讲讲**HeyGen的"历史"** 。

很多人只知道HeyGen是国外公司做的，其实并不知道，这家公司的创始人是中国人。

而且这家公司的前身是中国公司，叫诗云科技（深圳）有限公司。


![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2Fl2VB7h1M5Nawue44MgicNpmJI72q6fwiclJtKwMfZTrdW6wz3gnMeOJQposL7JpObs0g9GTRPaicS2qOuDI6ws9HQ%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

从23年12月份决定解散中国团队到彻底关停注销国内的运营主体，**只用了一个半月的时间。**

乌克兰网红Olga Loiek，也因为HeyGen"火爆"网络，成为了**欺骗性的"网络狂欢"** 中的一部分。


![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2Fl2VB7h1M5Nawue44MgicNpmJI72q6fwicl3ibaiaCZax4ZicWvQeibtLKVFS0vyOP1uB10WVfl33ibn18jPlqQLWA2L8g%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

**更震惊的是，就在注销前不久，HeyGen刚宣布获得新一轮融资**，高调吹嘘前景，却旋即作出撤离中国市场的决定。

坊间传闻，HeyGen此举很可能是为了规避中国日益严格的AI监管和数据安全法规，将业务和数据中心完全转移出境，以逃避国内监管。

中国用户的数据，人脸、语音、个人资料，就这样被带着离开了。

**反观开源项目HeyGem**


![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2Fl2VB7h1M5Nawue44MgicNpmJI72q6fwicluXrdcib0iaedQSUU3lBWxytiatu4jGzLkzuSgnPelYt30uSokwMjQQzibg%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

曾经十几块一分钟的克隆数字人，现在直接免费放给大家去用，我还有点不习惯。

其实也**不只是免费，而且还降低了使用门槛，消费级显卡就能跑。**

曾几何时，普通消费者哪能在自家电脑跑得动这种模型，算力成本高的要命，只有大企业才用得起。

**便宜，效果却很能打！**

硅基智能开源HeyGem之后，还有不少网友说，这肯定不是真开源吧，还有人说，我怎么没看见模型。

其实是真的把模型开源了，只是包在容器里了，真正懂容器的人是可以把模型拿出来的。

**HeyGem使用教程**


两个使用方式，一个是本地运行，需要一定的代码基础，一个是在云上使用，小白也完全没问题。

**本地运行**


1、安装docker、Nodejs 18

2、拉取镜像


    国内镜像 docker pull crpi-87t8jkvf79rj7hal.cn-huhehaote.personal.cr.aliyuncs.com/modelers_park/fun-asr:latest` docker pull crpi-87t8jkvf79rj7hal.cn-huhehaote.personal.cr.aliyuncs.com/modelers_park/fish-speech-ziming:latest docker pull crpi-87t8jkvf79rj7hal.cn-huhehaote.personal.cr.aliyuncs.com/modelers_park/heygem.ai:latest 
    原镜像 docker pull guiji2025/fun-asr docker pull guiji2025/fish-speech-ziming docker pull guiji2025/heygem.ai


3、运行

* docker-compose.yml文件在/deploy目录下。

* 在/deploy目录执行docker-compose up -d

4、打开客户端

* 客户端安装包已存放在/Releases目录下

* 双击HeyGem-x.x.x-setup.exe即可安装

**中国AI开源当自强**  


HeyGen企图通过投诉来让开源的HeyGem下架，还是停留在**美国的一些公司"闭源优越"的落后逻辑。**   

中国的大企业做开源AI，逆势而行，早晚会打破AI行业的技术垄断。

**从硅基智能的HeyGem、到杭州的DeepSeek，再到武汉的Manus，** 还有众多由中国的开发者开源的模型，正逐渐的火爆全球。

中国人的开源AI，也在慢慢让全世界人**接受、使用、赞美。**

**这些所谓的技术壁垒，早晚都会被开源打破，建立起一个好的开源生态。**
