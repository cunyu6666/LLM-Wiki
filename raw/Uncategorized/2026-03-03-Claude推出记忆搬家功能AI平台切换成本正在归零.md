# Claude推出记忆搬家功能，AI平台切换成本正在归零

[mp.weixin.qq.com](https://mp.weixin.qq.com/s?__biz=MzE5ODY5MDU4Mw==&mid=2247484893&idx=1&sn=94757265c1350cc930b8cc9a7d5777b6&chksm=97a1a5c442c078f846181c9858fe6299219fda2e4dd051befe9f74b28a5fc613b2b72776dc83&mpshare=1&scene=1&srcid=0303pMmnElYiAyaKkS3Xy6kG&sharer_shareinfo=8f06fa0fdbd1364348b99fbeefc6d752&sharer_shareinfo_first=8f06fa0fdbd1364348b99fbeefc6d752)鲁工 AI编程实验室

大家好，我是鲁工。

最近AI圈最热的事，除了各家模型的你追我赶，就是海外的QuitGPT运动了。前两天在X上看到Claude趁着这波热度上线了一个Import Memory功能，能让用户把在ChatGPT或其他第三方AI助手上积累的记忆数据，搬到Claude里来。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FMHVsz9lwuhUaic3x7jlvmOlFJfKuqSPDgdtULzkcE7ghO1g5AX0oeG1Adt336hkb1z3giawo9DJOvRGeNgKcztia0iaMm0KeuEhZekxZmKwA24Q%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D0)

我的第一感觉就是：这招够狠。

简单交代下最近背景。前两天，OpenAI宣布和美国五角大楼达成合作，允许在机密网络中部署模型。这件事在海外引发了巨大争议，#QuitGPT和#CancelChatGPT两个话题在社交媒体上迅速发酵。据QuitGPT.org的数据，已有超过150万人参与了抵制行动，包括取消订阅和卸载APP。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FMHVsz9lwuhXH19tic2cCoB3xgicBrhzsjz6OKqwKCKK69ImfyJh1LWbP220Mhqx5144ZUxCibNYK1uvRlHwJP90XicnSQ5WDjhar8HNyqOwIlvk%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D1)

与此形成对比的是，Anthropic此前明确拒绝了与五角大楼的类似合作。大量用户涌向了Claude，直接把Claude推到了美国App Store生产力类应用的第一名。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FMHVsz9lwuhUs5J3Mnnfvf39HDbbSjtVHibQtmicRagQ55QomDTL7NPR7ic0iafWcGaoF0TwzTOv3qsPtmGicVgQQXG9FnuwraKLxyen6wPmzGibJ0%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D2)

就在这个节骨眼上，Anthropic推出了Import Memory。时机拿捏得相当精准。对OpenAI来说多少有点杀人诛心。

说完背景，再看这个功能到底怎么用。

整个操作其实非常简单，核心就两步。第一步，打开Claude的记忆导入页面：

https://claude.com/import-memory

页面上Claude会给你一段提示词，直接复制。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FMHVsz9lwuhUiaiaTbgrZHibMKhKtkI1ukqQhHuDo3eV2bqNrLM9B7wqTSdQyNBsk1rjsPoiaPGCZKK6zOw1IBgo0HwLaLxplLCfxX7V9Bt0BsIc%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D3)

提示词如下：

Export all of my stored memories and any context you've learned about me from past conversations. Preserve my words verbatim where possible, especially for instructions and preferences.

## Categories (output in this order):

1. \*\*Instructions\*\*: Rules I've explicitly asked you to follow going forward --- tone, format, style, "always do X", "never do Y", and corrections to your behavior. Only include rules from stored memories, not from conversations.

2. \*\*Identity\*\*: Name, age, location, education, family, relationships, languages, and personal interests.

3. \*\*Career\*\*: Current and past roles, companies, and general skill areas.

4. \*\*Projects\*\*: Projects I meaningfully built or committed to. Ideally ONE entry per project. Include what it does, current status, and any key decisions. Use the project name or a short descriptor as the first words of the entry.

5. \*\*Preferences\*\*: Opinions, tastes, and working-style preferences that apply broadly.

## Format:

Use section headers for each category. Within each category, list one entry per line, sorted by oldest date first. Format each line as:

\[YYYY-MM-DD\] - Entry content here

If no date is known, use \[unknown\] instead.

## Output:

- Wrap the entire export in a single code block for easy copying.

- After the code block, state whether this is the complete set or if more remain.

第二步，把这段提示词粘贴到你正在用的AI助手里（ChatGPT、Gemini、Copilot都行），它会把存储的关于你的所有记忆以文本形式输出。然后把这段文本复制回来，粘贴到Claude的记忆设置里，点"Add to memory"就完事了。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FMHVsz9lwuhVLNF61MZslPMVr8mY5hRTpVF3x9lzZSp3prV6tfKyl8hoib6rU3hM0fEy1XokJ5sXKHrmVhKIiaNbI91gx2tX4ZWPVW8J9Sh0w4%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D4)

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FMHVsz9lwuhVN3xRAajI5HFjydrw0fDPz6nbzic2ziahD7iboqbZosrX5qO9qwUgLjZr2ozGdgXBzo96IQ1KkCG9LuXHexgsyzTUL78AWiaoywE8%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D5)

如果你用的是ChatGPT，还有一条更直接的路径：进入Settings，找到Personalization下面的Manage Memories，直接复制里面的记忆条目。

这段提示词的设计其实挺讲究的。它要求AI把你的所有记忆以代码块格式输出，逐字保留原始表述，不允许总结或省略。涵盖的内容包括你给AI设定的回复偏好（语气、格式、风格）、个人信息（姓名、职业、兴趣）、正在做的项目、常用的技术栈，以及你对AI行为做过的各种修正。

说白了就是用一段精心设计的提示词，绕过了各平台的API限制，直接通过对话的方式把你的数字档案提取出来。有点提示词攻击的味道了。

不过这里有几个需要注意的地方。

能搬走的是记忆和偏好，搬不走的是对话历史、文件附件、以及你在ChatGPT上创建的自定义GPTs配置。所以更准确的说法是记忆迁移，不是完整搬家。

导入的记忆需要最多24小时才能完全生效。因为Claude的记忆系统采用每日综合周期（daily synthesis cycle），简单来说就是每天统一整理一次记忆，不是实时处理。不过你直接在记忆设置里手动编辑的内容可以立即生效，不用等。

另外导入的记忆是和Claude现有记忆合并的，不会覆盖。已经用了一段时间Claude的用户不用担心原来的记忆被冲掉。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FMHVsz9lwuhXadG4rsVrPthKibrk0ibd1A7ZBRl3og9G2qNHbnsX4cQGErzGicmtrJFiatn15EuHYqTqOhia3vsyu2qThdJmVKln8jgP3cqWZLOGE%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D6)

关于隐私这块，Claude的记忆数据经过加密存储，官方明确承诺不用于模型训练，而且用户可以随时导出自己的完整记忆。这个双向设计（既能导入也能导出）在我看来是个加分项，至少说明Anthropic在数据可携带性这件事上是认真的。

目前这个功能对所有Claude付费用户开放，包括Pro、Max、Team和Enterprise。需要提一句，官方也注明了这个功能目前仍在实验阶段，不一定总能成功整合导入的记忆。

不过根据一些实测反馈，ChatGPT在对这段提取记忆的提示词时，导出的内容可能比较有限，甚至返回空结果。这可能和每个人的账户使用情况有关，也有人猜测是OpenAI做了某种限制。相比之下，Gemini面对同样的提示词可以正常输出基础信息。

过去我们换社交平台，粉丝和内容带不走。换AI工具也是同样的道理，你在ChatGPT上用了半年一年，它知道你是谁、你喜欢什么风格的回答、你在做什么项目，这些积累就是切换成本，也是平台留住你的筹码。

Claude这个功能，实质上就是在把切换成本往下压，顺带恶心一下OpenAI。复制粘贴一下，新助手就大致知道你是谁了，不用从零开始重新调教。

AI时代，每个人都会使用多家AI产品，记忆能在不同平台之间流动，意味着可以更灵活地根据不同任务选择最合适的工具，对我们用户而言肯定是利好。
