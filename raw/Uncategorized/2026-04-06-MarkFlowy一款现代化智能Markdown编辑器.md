# MarkFlowy：一款现代化智能Markdown编辑器

MarkFlowy 是一款免费开源、轻量级、现代化的 Markdown 编辑器，提供了强大的 AI 辅助功能。
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FlFpnG1ib0JBUD90jmNIKu1v0C9wUsr7VPlxlhDcKliaQYKVP4dfaul6KDjAhgef7Er9LBefXW5zwxQ8VX5yVBLib8jUoXzRVGZ8qmaia8l8iaxx8%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%26watermark%3D1%23imgIndex%3D0)

MarkFlowy 采用 Rust、Typescript 语言开发，遵循 AGPL 3.0 开源协议，代码托管在 GitHub：

https://github.com/drl990114/MarkFlowy

## 功能特性

*
  • **跨平台** ：MarkFlowy 支持 Windows、Linux 以及 macOS 平台。
*
  • **内置 AI 集成** ：内置 Copilot、文本生成、翻译、摘要功能，并且支持 DeepSeek、ChatGPT、Ollama、Gemini 等多种主流大语言模型。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FlFpnG1ib0JBWPrkulD7O8Ztjy66niaUMFvXLHMvH27zgXdF3nZfhMxkoZWeDuXIUFAnJJFfFwlibl9uOicna0psBU5dSpUCeYWTkmrK4HPrOIiak%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%26watermark%3D1%23imgIndex%3D1)

*
  • **轻量级** ：基于 tauri 矿机，拥有小于 20MB 的体积和更好的性能。
*
  • **多种编辑模式** ：MarkFlowy 使用 prosemirror 作为编辑器核心，用户体验好，扩展性高；支持支持所见即所得（WYSIWYG）、源码模式、预览模式。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FlFpnG1ib0JBWiatIXdetFzRqTibya81zn0OvU6PLonhHibJoJn5ichtDmRYicVJe7RQpULwbEdG6LvJWFjSwMwicFvcXjbuIOMvjdTJcVvwc7nHVQI%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%26watermark%3D1%23imgIndex%3D2)

*
  • **多种文件格式** ：除了 Markdown 之外，还可以支持编辑 JSON、TXT 等文件类型。
*
  • **图片处理** ：粘贴图片到 MarkFlowy 时可以选择粘贴到指定的路径或者转换为 base64。
*
  • **文件管理** ：提供功能强大的文件树，支持拖拽移动、全局搜索等常用功能。
*
  • **自定义快捷键** ：支持自定义快捷键，满足个性化需求。
*
  • **自定义主题** ：提供多种种内置主题，并且支持自定义和分享主题。
*
  • **导出文档** ：可以将文档导出为 HTML 或者图片。
*
  • **简繁转换** ：提供中文简体字与繁体字的自动转换功能。
*
  • **多语言支持** ：支持中文、英文等多种语言。
