### Figma 新增字符组合，转换为智能符号

特殊符号输入不用再去小键盘找了，Figma 支持将常见的字符组合自动转换为智能符号并适用于所有字体。

当你输入以下字符组合：(c)、(R)、(TM)、-\>、\<-、vv、\^\^、\[ \] 时，Figma 会自动将其转换为对应的智能符号：©️、®️、™️、→、←、↓、↑、▢。只要开启了 **「偏好设置」\> 「使用智能引号/符号」** 选项，该功能就会生效。此外，这一功能也已扩展至 FigJam 和 Slides，帮助你更高效地完成设计工作。

注意符号输入顺序，如果先输入两个括号（），再插入字母 R ，自动转换不会生效。

*https://www.figma.com/release-notes/*

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_gif%2FGZVDLRyzI6plNZRcvyfDLc9PuaibibSbz3BibXh3OTrAC5wVQU7GtuYZm1YedVw0oN2cwAic17HMZHKASVDalR3Jqg%2F640%3Fwx_fmt%3Dgif%26from%3Dappmsg)

## 🏆 产品推荐

### 分享你的「代码」

AI 时代下每个人都能比较轻松的完成一个网站、一个游戏的制作，但大部分情况代码只能在本地运行无法分享给其他人，将你的代码文件上传到这个网站，就能生成一个自己程序的访问地址分享出去。

*https://www.yourware.so/*
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FGZVDLRyzI6plNZRcvyfDLc9PuaibibSbz3o8v0YkrR4NicT0cyMw1VXib6oAYHATQuXIA76BYucRKWUvs1kWGoX5wg%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

### 运动曲线生成器

Easing Wizard 是一款在线工具，可轻松生成和自定义 CSS 缓动函数。可以生成 CSS 和 Tailwind CSS 代码，支持一键复制代码使用。有不少类似功能的网站，但是 Easing Wizard 属于视觉和交互都很棒的

*https://easingwizard.com/*
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FGZVDLRyzI6plNZRcvyfDLc9PuaibibSbz3hwGd04o4ib3XHdl9NPD9jcPiaRScRTXQzpEYibic6qrVH5xMU0uBibZC1icg%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)
