又是一年一度哄自己改作品集的时候了🥲 不知道是不是因为自己是个喜新厌旧的人呢，感觉今年的审美已经和去年的审美不一样了，作品集越看越不顺眼，所以打算大改特改。  
之前一直都是用 Webflow 做设计作品集，看了一眼 Framer 感觉学习曲线太高了，也不喜欢过于模板式的设计。最近上手 Claude Code 就决定用 AI 自己设计一个。  
前端： 因为脑海里已经有大概想法了，所以基本上全程没开 Figma，除非有几个方向想比较，才会快速画个图，截图让 Claude 直接帮我做出来。用 AI 做网页真的比自己在 Framer / Webflow 研究简单太多了！很多交互用几句话描述就能实现，省了大量时间。  

后端： 后端用的是 Notion，建了一个 table database 把内容写进去，再把 API 连接到网页的 .env 文件，这样不用把文字 hard code 进代码里，随时想改就改，非常方便。缺点是 layout 比较死板，想加设计元素都要自己 define 或用 workaround。  

图片： 图片存放在 Cloudflare R2。Notion 上的图片有时限，放在网页上会显示不出来，所以图片全部要丢上 Cloudflare，再把 link 复制进 Notion，才能正常显示。  

Skills： 在 Claude Code 安装了 skills.sh，装上了 seo-audit 和 audit-website。建议对于不太了解的领域或者步骤复杂的（对我来说就是 SEO 和 accessibility），都用 skills 来帮你看，可以省下不少时间。  

自己做网页步骤实在太多了。用模板只要把作品和图片丢进去就好，自己做的话排版、交互、states 全部都要自己定义......很累很累。API、GitHub、Vercel 要自己研究，图片太大还要自己压缩🫠 不过算是一个很好的锻炼机会吧。  

虽然最后成果没有那些设计大佬那么 fancy，但整体还是很满意的。
