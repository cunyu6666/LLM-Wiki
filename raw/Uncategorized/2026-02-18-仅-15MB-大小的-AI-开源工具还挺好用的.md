# 仅 1.5MB 大小的 AI 开源工具，还挺好用的。

[mp.weixin.qq.com](https://mp.weixin.qq.com/s?__biz=MzAxOTcxNTIwNQ==&mid=2457992594&idx=1&sn=0b73d4d188d34057e86550cf92c7dddf&chksm=8d7c40faf8a675d44109e0cbd750433cf1cc7e4afcc8c2a5d41fef1401666324bc1663300db3&mpshare=1&scene=1&srcid=0218gHzVKJ0J2Pp4XRcgBfg6&sharer_shareinfo=33ecc3a099c97a0025eabeb70b9fefce&sharer_shareinfo_first=33ecc3a099c97a0025eabeb70b9fefce)小 G GitHubDaily

这两年，「超级个体」和「知识分享」全面爆发，无论是录制教程、做播客还是搞直播，大家对出镜表达的需求直线上升。

但不少人只要一面对镜头，就容易遇到忘词的尴尬。为了不卡壳去偷瞄备忘录，眼神又显得不太自然。

虽然也有一些提词器，但大部分都有限制，有些甚至还要我们把底稿上传到云端，这难免会担心数据被泄露。

直到最近在 GitHub 上，偶然发现了一款名为 **Textream** 的开源工具，正好能解决我们的需求。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_gif%2FsnxIHWuwQonZPtPfLdyCSYQbeKN32whps4oESyF4FoLX1PU71tIicRiboeR6Gz1LwqwHK9icxTgsEMia2ibvtZuaU1Daa0tx4T7Y2iaRGb12ibEVzg%2F640%3Fwx_fmt%3Dgif%26from%3Dappmsg%23imgIndex%3D0)

这是一款完全开源免费的 macOS 桌面级提词器，其最大核心功能，就是自带的"语音追踪"模式。

可以直接调用苹果设备本地的语音识别引擎，全程离线运行，不需要连网。

而且不管我们语速有多快，它都能做到读到哪就高亮到哪实时追踪。

如果中途停顿，高亮也会立刻跟着停下来。这就意味着，它能很好地顺应我们讲话的自然节奏。

在显示交互上，它也做得非常好，完美融合电脑屏幕顶部的 "灵动岛"，紧紧贴在刘海屏下方。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FsnxIHWuwQonsKkvzJjtNHkOOev8Sv41zZpVcySHHrIiav8NOcD95pj8vOGbxxpQysLibA8Pib0OAqSLqib9xC3ORiaNUnIWXe6CBCRQrCZkLTHVc%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D1)

当然，也可以变成带有毛玻璃效果的悬浮窗，随意拖到屏幕的任何角落。

更有意思的是，开启屏幕共享时，这些悬浮窗对观众是隐形的。

我们可以大大方方地照着念，不用担心穿帮。平时开会用的 PPT 备注，也能直接丢进去提取成提示词。

甚至还能拿手机扫个码，把提词界面同步到局域网内的移动设备上。

更值得一提的是，这款工具对中文的支持也十分友好，大家可以放心试试。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FsnxIHWuwQok1oayCqQ1TVO7Nua1wPxxm2kicl732iaE6KFhLcP3jicXuzrjUF7ayFhS5Hoy4Jf9r1zVSLdkUwD8icxbiajRJfyShpy91ZE6h2Vr4%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D2)

想要上手十分简单，如果电脑上装了 Homebrew，只需在终端跑下面这行代码就能搞定。

    brew install f/textream/textream


也可以直接去项目的 Releases 页面下载 dmg 开箱即用的安装包。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FsnxIHWuwQomaiaqve9lfp7KIrIIibRhxfoDHQ2W5iaxhl9TibByvGvnhFr4HniaJWZmyL7O8n2oAAdCnOz2xhwpBxbkxoNGBpg9Y7TMZjdOzpOLI%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D3)

不过这里有个硬性门槛，它依赖苹果系统的新特性，所以电脑必须升级到 macOS 15 才能正常运行。

另外，第一次打开可能会被系统拦截。只需在终端运行一次解除隔离的命令，或者右键点击应用选择打开即可。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FsnxIHWuwQolrgH7cKsQJL138UGx4hRZlkWgXLM8b5eFlvmhAyxFtCWhC1MgGjhpeFj9ErP7icKcuobhFiblx0QN0wQNAPZkdqkf7lPxACE4H4%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D4)

有了这款工具，我们在镜头面前就能自然流畅的表达，也不需要花钱购买其他提词器。

只需一台普通电脑，就能在完全离线情况下使用，有效保护我们数据，并完成流畅的语音内容输出。

这也侧面表达了，真正优秀生产力工具，不需要盲目靠云端大模型的堆砌。

像这种巧妙利用本地设备的 AI 算力的小工具，将会不断涌现，越来越多，

随着这类开源项目的不断涌现，高质量内容的创作门槛会变得越来越低。

GitHub 项目地址：https://github.com/f/textream
