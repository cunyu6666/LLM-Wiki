****copyparty****


今天要认识的是copyparty

拖一下文件，瞬间上传，这就是 copy party 一个 Python 文件搞定一切。

这个文件秒传神器，100%完全开源。

GitHub 一周疯涨17K star 手机、电脑、树莓派全部兼容，它不只是文件服务器，更是完整的数据管理中心，磁盘监控、权限控制、并发上传、状态追踪、企业级功能全都有。

看这专业级界面，不光管文件，还能管项目，从个人播客到企业代码库，可用空间实时监控，上传队列状态一目了然，上传到一半断网没关系，重连秒恢复。

这就是为极客量身定制的暴力美学。

看这操作有多丝滑，直接拖文件到搜索框并发上传，自动优化磁盘空间实时监控，从只读到管理员权限控制精确到位。


https://github.com/9001/copyparty

**PandaWiki**

今天要认识的是PandaWiki
GitHub上这个开源的知识管理平台最近火了。

一键构建文档、手册、FAQ、博客系统、AI协作、问答、搜索样样精通，在GitHub上迅速拿下6K star。

不仅如此，Markdown编辑、多人协作、版本控制、权限管理、全文搜索、数据导出、再加AI辅助，这就是降维打击。

那么部署有多简单？ROOT登录服务器，一条命令搞定，几分钟后直接可用。

安装好后直接用左边目录结构，一目了然。

Markdown:编辑实时预览，多人同时编辑也不会冲突。

搜一下马上找到，还有更多功能，权限管理，想怎么设就怎么设，版本记录随时能回退，而且完全开源，代码想看就看，企业用也完全够。


https://github.com/chaitin/PandaWiki

****workout-cool****

今天要认识的是workout-cool
它是一个开源免费的健身平台，既可以本地部署，也可以直接打开网页使用，选好器械和部位，训练计划立刻生成，每个动作都有详细的视频，跟着做就行，动作演示清晰详细。

跟着视频练就行，再也不怕动作不标准。

练完还能打卡复盘，训练数据全记录。

最重要的是完全免费，界面清爽无广告，赶快去试试吧。

本地安装也简单，克隆仓库、安装依赖、启动开发环境localhost3000就可以访问。
