# 这个开源项目，想要永远记住你。

[mp.weixin.qq.com](https://mp.weixin.qq.com/s?__biz=MzkxMzUyODQ3OA==&mid=2247486399&idx=1&sn=196f1752ee83725dfa293f1efa631d65&chksm=c029e0c9b9e710b42d62b4e3479a2e124847e34c0e26a6b622509b60452b8a0d822a07976154&mpshare=1&scene=1&srcid=0816FbxXcKODELfHY6yaTgcK&sharer_shareinfo=ac4eaf697dafeef08e7d767243a9fa79&sharer_shareinfo_first=ac4eaf697dafeef08e7d767243a9fa79)银海inhai AI产品银海

Hi，我是银海。

你身边的AI产品经理，陪你解决每一个真实问题。

不知道大家有没有这种感觉，和 AI 聊天最让人抓狂的，就是它总是"失忆"。

我刚花了十分钟把背景跟它讲得清清楚楚，开个新窗口之后，**它就啥也不记得了，却完全忘了你们聊过什么。**

后来我找到了一个能够将你和模型对话的记忆保存下来的开源的项目，叫做 **memU，不仅能够记得你是谁，还能记住你们聊过的每一句有用的话、做过的每个决定，并且在后续的对话里主动帮你接着往下做。**

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmZ1Kw7NUwAYToyP2g3kO9OhicWPAzZbHx9yJMnIGnh5AvibPe96Viciczeg%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

GitHub地址：https://github.com/NevaMind-AI/memU

memU 就像是给 AI 伴侣加了一层专属的"聪明记忆"，反应快、记得准，还不怎么费资源。**跟那些通用型的记忆系统不一样，它是专门为那些特别在意聊天上下文、懂你关系变化、还能自己学着适应你的 AI 伴侣准备的。**

这个就是我测试出来的效果，它把不同角色、不同身份的对话记录，**都分门别类地存进了它的"记忆空间"里。**

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmgT7WzsNCTZxTvyTRcLibZRzx5vnrteShe8rpVamkSbArRI3FDvtK0kg%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

这样一来，这些内容就不会聊完就消失，它能够被长期保存下来，等我之后要**复盘、引用、甚至二次创作的时候，都能随时拿出来用。**

下面是 deepwiki 对 memU 项目核心组件架构图的解读，如果你想深入了解 memU 的代码逻辑和实现方式，推荐你直接点这个链接看看：https://deepwiki.com/NevaMind-AI/memU

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmpN7U7TdGUAu6ne1M5hjYQ26bdKeHDs8QU3GmLs5YUgtkocAsTXNfng%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

**真心说，这个开源项目真的值得去点个 Star。**

我自己平时也折腾过不少记忆类的产品，像 mem0、zep、memobase 这些都用过，结果对比了一下 memU 的竞品跑分，发现它的成绩是真的猛，不光是速度快、准确度高，连成本也压得很低，完全是那种"看一眼数据就想试试"的水平。

拿着行业里几个有代表性的记忆系统来对比，memU 的平均准确率高达 **92%...每个都高出了一大截。**

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmztZy4cDZW7hvg9EhbAT3qpzJME60kM1P4ZcukX5HbgeTcCyL6Cw62w%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

**虽然数据很牛逼，但我认为有记忆的 AI 的价值是每次对话都不是从零开始，而是在不断的累积。**

就如下面这张图所呈现，memU 会把你跟它聊过的话、做过的事，甚至发过的图片视频，自动整理成一份份小档案，比如"你喜欢熬夜""昨天通宵了""记得晚上 11 点提醒休息"这种，**然后这些档案还能互相关联，越用越懂你，还会自己去总结、补全信息，慢慢形成一个真正了解你的 AI 伙伴。**

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmeORf7F6gw6ENatQr0YFFcPYkhrlrDYu0hF4ATxiau61jcticP6e7zcrg%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

据 deepwiki 的介绍，MemU 的核心记忆系统走的是一种很特别的路子，并没有把记忆拆成一堆零散的向量，**而是像建文件夹一样，把它们整理成一份份有逻辑的智能文档。**

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmPy7jnDZZuXN4fU5gIA1tlhZFJlHwfGUeAFoia0rE5iaQh0wzWt5NDEuA%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

这样一来，不光**检索的时候更准** ，还能让**记忆的结构更自然、更像真人的思维方式。**

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmldpMv5obVZhxgUQVJS7r16Sc6pEd8DUz8Rs4xJWVLo4icY97ick7Q0kA%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

更重要的是，如果你马上想在项目中试试 memU ，用起来一点也不复杂，可以直接用它的云端版本，几分钟就能跑起来。

先去 https://app.memu.so 创建个账号。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmSBrIPfGrNV3KxmzE7ODiapc8ZQzoXxEDl8KYApMoIa4kjib5OqD0NLbg%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

然后到 https://app.memu.so/api-key/ 生成 API key。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmqWuPBuXTZ93wLACPia0Ma6nEA4CT7kgFB4WHJQ9eXWmgw08U3hibkbicw%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

接着打开终端去装个包。

    pip install memu-py


然后在代码里加几行。



        # Example usagefrom memu import MemuClient# Initializememu_client = MemuClient(    base_url="https://api.memu.so",     api_key=os.getenv("MEMU_API_KEY"))memu_client.memorize_conversation(    conversation=conversation_text, # Recommend longer conversation (~8000 tokens), see https://memu.pro/blog/memu-best-practice for details    user_id="user001",     user_name="User",     agent_id="assistant001",     agent_name="Assistant")



当然了，你也可以直接把 GitHub 上的项目代码拉下来，然后配合一些 AI Coding 工具，很快就能跑起来做几个自己的小案例。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmpRsN7GTjCGrTedb8eWjA4lU1Bjx325GaOUKk02mAw8MlGeoEO0NQpw%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

图中所示就已经把我的对话存储到memU的云盘上了。这样一来，不管是任务记录、项目进展，还是聊天故事，都能直接把相关的记忆保存到云上，用起来既方便又有成就感。

照着这个方法跑一次，你就能让自己的 AI 拥有长期记忆，我们可以在 memU 的云平台上看见已经将记忆同步存储啦！

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmbnC6SDp50Ca94Gh5xPHXC6j4MkXtra2pvP6r6kQg5lrCicd1nLICusw%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

你还可以随时检索你过去的记录，并把它们直接集成到自己的项目里。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FGkSBRc9GpQ4NL0DibcFwS6qgmcv9XsQXmEoiaYcmuC6Hc7Hia90RvSNbD9lo654Qr2C0QneDzeDwvY5EwKciarYN5w%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

如果你对记忆存储也有需求，不妨先找个能持续几天的小任务试试，比如每天和它聊聊你的学习计划、让它帮你盯着一个项目的进度，或者干脆一起编一个连载故事。

等到第三天、第四天的时候，你会突然发现，**它不仅能顺着你前天的话往下接，还会提醒你落下的事，甚至帮你补上当时没说完的思路。**

那一刻，你可能就会有点恍惚：**原来 AI 也能像老朋友一样，记得你的昨天、关心你的今天、参与你的明天。**

而我们平时用的**大部分 AI 其实是健忘的，上一句话还没热乎，就被它忘在了风里。**

**记忆，才是让 AI 变得"有温度"的关键。**

所以我真心建议，你去 GitHub 看看它，也欢迎给这个开源项目它点个 Star。  

项目地址：https://github.com/NevaMind-AI/memU

另外我还有一个大胆的想法，如果能够实现一个浏览器插件，在跟所有 Chat 产品对话过程中，自动把记忆存到memU里面，那将会是一件多么酷的事儿！期待！

我们都在支持一个事儿：**让 AI 记得住你、记得住我们。**

这种支持，也许有一天会回到你身上。
