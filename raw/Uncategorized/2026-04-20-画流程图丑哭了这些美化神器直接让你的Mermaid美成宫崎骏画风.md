平时做开发，写 markdown 文档， **Mermaid.js 基本躲不开** ，它是一个非常成功的开源 JavaScript 库。

用一句话概括就是：用写代码的方式画图。

流程图、时序图、类图、甘特图、ER 图......不用拖拽，不用对齐，不用和 Visio 较劲。

只要像写 Markdown 一样敲几行：

    graph LR    A --> B --> C


![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FvqlbVFl5Jn0UUcVxWhoBtvt9V1iaeu285VosJoEicGkT31VBBfGrM0XevxpU8nzpyKkqygibsPNXvvmy7ibIhvMwqQ%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D0)

这样，一张结构清晰的图就出来了。

也正因为这种代码即图表的体验，Mermaid.js 从 2015 年左右一路火到现在，**GitHub 已经 85k+ stars** ，

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FvqlbVFl5Jn0UUcVxWhoBtvt9V1iaeu285JIgpcQialLRV5hvyZibx2eA3dFVuRgsLveXzIREOhoObPQepJYbKW7DA%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D1)

对开发者、产品经理、技术作者来说，Mermaid 几乎是刚需工具：

* 技术文档标配

* Markdown 编辑器内置功能

* Obsidian、Notion、Typora 的常驻嘉宾

1、**beautiful-mermaid**

**beautiful-mermaid 是一个用** **TypeScript/JavaScript** 编写的开源库，目的是将 **Mermaid 图表** 渲染成 **美观的 SVG 图像** 或 **基于文本的 ASCII/Unicode 图形** 。

**beautiful-mermaid 被设计为** **轻量、快速、可主题化** ，适合在浏览器、Node.js、终端、AI 工具等各种 JavaScript 环境中运行，而不依赖 DOM

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FEYxI5IfBe8WliaQX0I9s2eibZ3ukjEwQeIGY7QTffKxv7Aicmuh41ojLoVbYib2X41HJqYwumfpNcgkib6iafXo7bhJaQEiawv8Ws6ja0NPibtrJoPU%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D5)

开源地址：https://github.com/lukilabs/beautiful-mermaid

安装命令：  

    npm install beautiful-mermaid


生成 SVG：

    import { renderMermaid } from 'beautiful-mermaid'const svg = await renderMermaid(`  graph TD    A[Start] --> B{Decision}    B -->|Yes| C[Action]    B -->|No| D[End]`)


生成 ASCII：

    import { renderMermaidAscii } from 'beautiful-mermaid'const ascii = renderMermaidAscii(`graph LR; A --> B --> C`)


浏览器环境使用（CDN）：

    <script src="https://unpkg.com/beautiful-mermaid/dist/beautiful-mermaid.browser.global.js"></script><script>  const { renderMermaid, THEMES } = beautifulMermaid;  renderMermaid('graph TD; A-->B').then(svg => { ... });</script>

2、Pretty Mermaid Skills

**Pretty-mermaid-skills 是一个围绕 Mermaid 图表渲染能力构建的开源 AI Skill 项目，目的是让 AI 工具（如 Claude、Cursor、ChatGPT 等）** **自动生成美观的 Mermaid 图表** ，输出格式支持 **SVG（矢量图）和 ASCII（终端字符图）** 。

**Pretty-mermaid-skills 是一个** **Mermaid 渲染技能（skill）** ，专门用于给智能助理／AI 工作流**赋能** ：

* 当 AI 收到生成图表的请求时，它可以调用这个 skill 去渲染 Mermaid 图。

* 渲染时自动应用漂亮的主题、美化样式和输出格式，而不需要用户手动配置细节。

也就是说，它把已有的 **beautiful-mermaid** 美化引擎封装成了一个 **AI 可调用的插件/技能接口** 。

开源地址：https://github.com/imxv/Pretty-mermaid-skills  

安装命令：

    npx skills add https://github.com/imxv/pretty-mermaid-skills --skill pretty-mermaid

3、Modern Mermaid

Modern Mermaid 是一个基于 React 的 Web 编辑器，构建在 Mermaid.js核心之上。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_jpg%2FvqlbVFl5Jn0UUcVxWhoBtvt9V1iaeu285tqcFxWPgkQCibicv4PvpWl81zG3RkiboAW2tQ03O6v9QyHW1ZYFibJOfdA%2F640%3Fwx_fmt%3Dwebp%26from%3Dappmsg%23imgIndex%3D6)

Modern Mermaid 不是重新实现 Mermaid 语法，而是提供了一个完整的图形化界面，让用户更舒适地编写、预览和导出 Mermaid 图表。

Mermaid 还是那个 Mermaid，但编辑器全面现代化升级。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FvqlbVFl5Jn0UUcVxWhoBtvt9V1iaeu285XLEG6uCqU7Bib9ECKwSX98fm352lYOWrxCiaaTf1Gg0r6WOSBebZt5MQ%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D7)

虽然目前 star 只有两百多（还很新），但开发者社区已经开始被夸：

* 终于不再是工程工具审美了

* 导出的图第一次能直接给设计师用

* 官方编辑器该学学这个 UI 了

**开源地址：https://github.com/gotoailab/modern_mermaid**

**在线体验：https://modern-mermaid.live/**

**核心特点**

* **背景自定义** ：告别单调，支持**渐变色** 、纯色以及各种底纹图案。

* **丝滑暗黑模式** ：全界面暗黑支持，还能随系统自动切换。

### 2、硬核编辑

* **智能编辑器** ：实时语法高亮、自动补全、行号显示------这哪是编辑器，简直就是图表界的 VS Code。

* **自由布局** ：编辑区和预览区的大小随你拉动，在大屏和笔记本上都能找到最舒服的姿势。

### 3、高清无码导出

* **超清画质** ：支持 **3 倍分辨率导出** ，哪怕放大 300% 依然锐利。

* **透明 PNG 专家** ：支持导出透明底 PNG，完美嵌入你的 PPT 或网页背景。

* **一键搬运** ：支持**剪贴板复制** （带背景或不带背景随你挑），甚至还能**批量导出** 。


![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FvqlbVFl5Jn0UUcVxWhoBtvt9V1iaeu285DDKTAGNDp8juAH1Ct6ibMIqetP5iabic7y5oicSIJibFLOrt704cngaya0Q%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D12)

### 4、不仅仅是预览

* **标注神器** ：自带箭头、矩形、圆形和文字标注工具，颜色会自动适配当前主题，让你的圈重点行为也变得优雅。

* **极致交互** ：支持平滑的**缩放 (Zoom) 与平移 (Pan)** ，还能开启全屏沉浸模式。

* **节点着色** ：对准节点**右键直接改色** ，再也不用在代码里苦哈哈地写 style 定义了。

* **多语言支持** ：界面自带 6 种语言支持。


![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FvqlbVFl5Jn0UUcVxWhoBtvt9V1iaeu285wCV2xLBfeliaibn12sLRiakrUKVG6icMxsgMwJnWukINNibDS3z2H1HbVkA%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg%23imgIndex%3D13)
