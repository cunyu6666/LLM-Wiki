## 专为大型代码库而生

Augment Agent 没有专注于"从零开始"的代码生成，而是帮助开发者在**大型现有代码库** 中导航和修改代码。

Augment Code CEO 表示："大多数编码 AI 工具专注于'从零到一'的代码生成。我们的目标是维护大型复杂系统------数据库、网络栈、存储------这些代码库经过多年演变，有数百名开发者协作参与。"
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fsz_mmbiz_png%2FwnIMIiaEIIrjT7Uiap90c3h7icic2oxmR8NQ9UpAqguEjRf77hwe5p6NSmTAVfIyzeR3JNvU9KRfpWBAZ9luice4pRQ%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg)

## **20万 token** 的上下文窗口

Augment Agent 最大亮点是其 **20万 token** 的上下文窗口，是大多数竞争对手的**两倍** 。这让它能处理跨越数百万行代码的项目，无需开发者手动组装相关上下文。

AI 处理包含数千万行代码的大型系统很困难。Augment 通过**实时采样技术** ，精确识别代码库中让Agent能有效工作的正确部分。

另一个优势是代码变更的**实时同步** 。当团队成员修改代码时，Augment Agent 立即获知这些变更，确保每个开发者看到最新代码。

## "记忆"功能适应个人编码风格

Augment Agent 的\*\*"记忆"功能\*\*会自动更新并在多次对话中保持，不断改进生成的代码，加快任务解决速度，并匹配你的代码风格和模式。

与认为编码纯粹是数学逻辑的观点不同，许多开发者非常关心代码的美学和结构。Augment Agent 能理解并适应这些个人偏好。

## 强大的工具集成能力

Augment 全面支持 **MCP** （模型上下文协议），提供各种工具和系统访问权限。早期用户已在使用 Vercel、Cloudflare 等基础设施 MCP 添加额外上下文、自动化工作流和调试生产问题。

Augment 还提供**原生工具集成** ，包括 GitHub、Jira、Confluence、Notion 和 Linear 等。这使开发者能连接整个软件开发生命周期中分散的上下文。

## 更多实用功能

Augment Agent 还包含多项实用功能：

*
  **代码检查点** ：自动跟踪变更，支持轻松回滚
*
  **多模态输入** ：分享截图、Figma 文件，帮助修复 bug 或实现 UI 元素
*
  **终端命令** ：允许 Agent 在终端运行命令，如 npm install、run dev 或与 Git 交互
*
  **自动模式** ：无需确认Agent的所有操作
