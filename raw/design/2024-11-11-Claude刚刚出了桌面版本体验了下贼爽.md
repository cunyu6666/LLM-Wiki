# Claude刚刚出了桌面版本，体验了下，贼爽

> 来源: [mp.weixin.qq.com](https://mp.weixin.qq.com/s?__biz=MzkxNzY0OTA4Mg==&mid=2247487254&idx=1) · 太阳彩虹 老码小张

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FoXqG8ETvAeleicnXIbI2rzLgy7wibHCPgtYr5NIaV1loExYDTWGeos7N3mwmSjmAUDWqpIzlcz95fdYVCo3FiaWicw%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg "null")

感觉国外的产品都是优先出 web 版本的，等市场做得足够大之后，才陆续考虑移动端，随后才是桌面端，不知道你是否还在用网页版的 Claude？那你可有点落伍了！Anthropic 近期宣布了多个新功能，让 Claude 的用户体验更上一层楼。无论是 Mac、Windows 还是移动设备，都有令人惊喜的更新，简直是为现代办公的"卷王"们量身定做。今天就带你看看这些新功能，顺便聊聊这些更新到底值不值得一试。
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FoXqG8ETvAeleicnXIbI2rzLgy7wibHCPgt1cObfSdTameFO4buJHl2r7rV2FFgibTPTVVic0svDvrze7Z9at73lmIA%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg "null")

#### 1. 桌面应用程序上线！Claude 终于不再依赖浏览器

下载地址：

https://claude.ai/download

想象一个场景：你正准备写一份报告，但 Chrome 已经被你打开了几十个标签页，Claude 的网页窗口被淹没在其中。每次要用 Claude，都得在浏览器里一通搜索，十分繁琐。好消息是，Claude 的 Mac 和 Windows 桌面应用程序现已正式上线！直接在桌面点击 Claude 图标，你可以立刻进入 Claude 的界面，免去繁琐的操作。而且也支持快捷键唤起，`ALT`+`SPACE` 一键唤起对话页面。
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FoXqG8ETvAeleicnXIbI2rzLgy7wibHCPgt5qqvy02vjXGj6l9BsK2olTdYtKicQDMIGQmT5xXYiaeVXn2ViaOibdMrCg%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg "null")

这个新功能特别适合日常需要频繁用 Claude 的职场达人。无论是文案编辑、代码编写还是数据分析，只需轻轻一点，Claude 就能迅速响应，不用再在浏览器里找来找去。你可以把它想象成一种"Claude 快速通道"，高效又省心。

#### 2. 移动端语音输入，让沟通更加便捷

同时，Claude 也支持移动端，可以直接在应用市场下载。
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FoXqG8ETvAeleicnXIbI2rzLgy7wibHCPgtxLOkCpGiaSbUpIAard62YcIqPz3ucRIURV3IbGRTKeN5yMQAn2DyCsw%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg "null")

一个场景是，你是否有过这样的体验？在公交车上、走路中，突然想到一段灵感，掏出手机想要记下来，结果光打字就手忙脚乱，更别提操作 Claude。现在，Claude 的 iOS 和 Android 应用新增了"语音输入"功能！通过语音输入，你可以直接对着手机"说话"，Claude 就能帮你把话转成文字，再继续帮你生成内容。

想象一下，你在开车、散步或运动时，也能把灵感记录下来，Claude 将成为你移动办公的好帮手。这项功能特别适合那些经常外出、需要即刻记录的用户，既节省时间，又让灵感不再流失。

#### 3. 支持视觉 PDF 分析，复杂内容也难不倒 Claude

提到 PDF，大部分人想到的可能就是文字内容，但实际上很多工作文档都包含复杂的图表、流程图，甚至是系统架构图。以前，Claude 只能读懂 PDF 的文本信息，遇到图片就"瞪眼发呆"。但这次更新后，Claude 终于能够理解 PDF 中的图片内容了，比如图表、结构图等！

这功能的应用范围相当广泛，比如：

- • **市场人员**：可以用 Claude 快速分析 PDF 格式的营销报告，看看不同渠道的 ROI（投资回报率）表现。

- • **工程师**：在复杂的技术文档中，Claude 可以解读系统架构图和流程图，帮助更好地理解整个系统。

- • **销售团队**：在客户策略报告里找到关键点，更加有针对性地准备客户方案。

- • **产品经理**：从用户反馈报告的图表中提炼常见的痛点和用户需求。

![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmmbiz.qpic.cn%2Fmmbiz_png%2FoXqG8ETvAeleicnXIbI2rzLgy7wibHCPgtNCY3BEGuOiae7QMcv5LZ0B5o4hS5wD1Sb2qjGhnkLqBXmfvMaoILojA%2F640%3Fwx_fmt%3Dpng%26from%3Dappmsg "null")

所以，对于需要处理含有复杂图像的文件，Claude 的这项视觉分析功能无疑是一大福音。虽然目前这项功能还处于预览阶段，但已经适用于所有 Claude.ai 订阅计划和 Anthropic API 用户。

### 一些看法：Claude 的这波更新值不值得一试？

如果你是 Claude 的老用户或是对 AI 辅助感兴趣的用户，这些新功能绝对值得体验一番。特别是桌面应用和视觉 PDF 分析，对于提高办公效率帮助很大。如果还没试过，那不妨现在就去 Claude 的官网下载试试吧！
