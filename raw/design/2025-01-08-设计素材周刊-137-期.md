# 设计素材周刊 137 期

> 来源: [moonvy.com](https://moonvy.com/blog/post/%E8%AE%BE%E8%AE%A1%E7%B4%A0%E6%9D%90%E5%91%A8%E5%88%8A/137/) · 四喜茶茶

## 🏆 产品推荐

### 批量设置文本行高插件：Pro Line Height

使用插件可以在 Figma 中批量为文本设置行高， 例如，可以将批次的行高设置为字符高度的 1.618 倍，然后将其四舍五入为 4 的倍数。非常实用的插件👍

<https://www.figma.com/community/plugin/1426841252851196012/pro-line-height>
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmoonvy.com%2Fblog%2F_assets%2F87ffb376e14e8e1a.png&valid=true)

### 快速生成设计系统插件：Figr Identity v2

Figr Identity 是一款 Figma 插件，帮助设计师创建和管理设计系统。它可以生成不同的样式和布局，支持自定义颜色和组件，让设计系统的过程更简单。

<https://www.figma.com/community/plugin/1350743748296105581/figr-identity-v2-generate-design-systems-quickly>
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmoonvy.com%2Fblog%2F_assets%2Ff502e23b2072f711.png&valid=true)

### 用图像生成文字 ASCII 艺术插画

超棒的插件，释放文本的魔力。

将任何设计转化为令人着迷的 ASCII 艺术杰作。无论您是想增添一丝怀旧气息，还是在数字领域释放您的创造力，Asciicore 都是您的首选工具。（需要使用等宽字体哦）

> 来源: [ASCII 艺术](https://zh.wikipedia.org/wiki/ASCII%E8%89%BA%E6%9C%AF) · 是一种使用 ASCII 字符（如字母、数字和符号）来创作图像或设计的艺术形式。它通过排列字符形成图案或图片，常用于早期计算机界面和文本文件中。

<https://www.figma.com/community/plugin/1238371742775577007/vga-asciicore-exe>
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmoonvy.com%2Fblog%2F_assets%2F1bcc900ac4d4426d.png&valid=true)

### 专为开发者打造的开源笔记应用

Kompad 是一款笔记应用，旨在提升用户的组织性和效率。它支持 MacOS、Windows 和 Linux 系统，并能实时同步数据，让你随时随地访问你的笔记。

<https://kompad.hudy9x.com/>
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmoonvy.com%2Fblog%2F_assets%2Fc75cde9a32b1546a.png&valid=true)

### CSS 圆锥曲线渐变代码

网站中漂亮的渐变提供 CSS 代码，点击复制即可使用，简单方便。

<https://www.conic.style/>
