## 📰 设计资讯

### 最近比较火的 谷歌 AI 图生图模型

目前为止尝试过的 AI 生图工具谷歌这款AI是最简单直接的，生成效果也很好，强烈推荐大家试试看。

Whisk 是 Google 实验室最新的生成图像实验，专注于快速视觉构思，而无需深入理解提示！只需添加几张图像作为方向参考（场景、主题、风格），Whisk 就会推荐一些图像供您继续完善。

Whisk 由 Google 的 Gemini（具有视觉理解功能的语言模型）和 Imagen 3（生成图像模型）协同工作提供支持。

可以将图画变成毛绒玩具？制作节日贺卡？或者卡通变巨物在现实中存在......

网站：<https://labs.google/fx/zh/tools/whisk/faq>

教程：[https://www.bilibili.com/video/BV1xDkRY1EWA/](https://www.bilibili.com/video/BV1xDkRY1EWA/?vd_source=3b9794030185d9477ec1a207ce025af4)

## 🏆 产品推荐

### 将文件、文档转换为 Markdown 格式

MarkItDown 是一款将各种文件转换为 Markdown 的实用程序，由微软出品。提供在线版本，打开即用非常方便的文档格式工具。

<https://markitdown.pro/>

<https://github.com/microsoft/markitdown>
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmoonvy.com%2Fblog%2F_assets%2Fa7f99b539f4d6b77.png&valid=true)

### 以节点的方式创建设计空间

使用节点式编辑器（Node-based Editor）的方式创建有意思的设计数据流，玩法很有意思，可以将图片、矢量内容、或者颜色，通过自定义的多节点创建一个有意思的数据流。这是一个开源的项目，假期有时间可以研究一下。

<https://bitspace.sh/>

<https://github.com/newfrgmnt/bitspace>
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmoonvy.com%2Fblog%2F_assets%2Fe2c7d4d0f8bde97c.png&valid=true)

### Mac 的剪贴板工具

保留你复制的所有内容，并在需要时快速访问你的 macOS 剪贴板历史记录。一次性收费工具。

<https://clipbook.app/>
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmoonvy.com%2Fblog%2F_assets%2Ff155411b15bbff1.png&valid=true)

### 在线字幕编辑工具：爱慕

支持上传视频自动生成字幕、提取字幕、字幕翻译等功能，可以在线免费使用。

<https://aimu.app/>
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmoonvy.com%2Fblog%2F_assets%2Fc3c2cf4ee97b3ec4.png&valid=true)

### 支持 3D 视图的图像美化工具

支持将图片添加边框、图标、设备框之外还有一个3D功能，可以将图片进行透视变形。是一款免费的在线工具。

<https://postspark.app/>
![](https://cubox.pro/c/filters:no_upscale()?imageUrl=https%3A%2F%2Fmoonvy.com%2Fblog%2F_assets%2Fddad6bd6f122f2b3.png&valid=true)

### 生成 GitHub 数据卡片

填入你的用户名，就可以生成 GitHub 数据卡片，还提供多种不同的外观样式，设计师的你可以分享给你的开发者朋友试试(*\^▽\^*)
