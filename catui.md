## Project Overview

基于 LLM 的个人知识库系统，面向设计师的知识管理与学习场景。从多源渠道（Cubox、抖音、小红书、设计周刊等）采集素材，经 Python 流水线自动分类、去重、增强，最终编译为结构化 Wiki 文章与设计资源索引。

**Tech Stack**: Python 3, Markdown, Karpathy LLM Wiki Skill
**Core Features**:
- 多源素材采集与自动分类入库
- LLM 驱动的知识编译（零散素材 → 结构化文章）
- 设计资源索引（图标/字体/插画/样机等 20 个分类，1500+ 条目）
